1. Using the loop-building strategy from the lessons, which of these are part of the loop mechanics?

    1. goal precondition
      The goal precondition creates variables for the goal, but isn't concerned with how the loop starts and stops.
    2. bounds precondition
      You create variables used by the bounds. If you don't do this, the loop will not start correct.
    3. loop bounds.
      The bounds determines when the loop will stop.
    4. goal operation
      This is used for calculating the correct output, but doesn't contribute to the loop's mechanics (starting and stopping correctly).
    5. advancing the loop
      The third part of making sure the loop runs correctly, is to make sure that at each iteration, it advances to the next.
    6. post condition
      Actions taken after the loop concludes, contribute to the loop's goal, but not to its mechanics.
