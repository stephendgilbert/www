1. Which of these statements is true?

    1. At the lowest level, all input and output is a stream of bytes flowing through your program.
    2. Programs that process streams of characters are called text filters.
    3. When you run a program, the operating system automatically opens four streams.
      It automatically opens three streams
    4. The operating system stream stderr is connected to your monitor by default.
    5. Filter programs read from input files and write to output files.
      From the program's perspective, it reads from stdin and writes to stdout.
    6. When running a filter program, you can send all output from cout to a file using the > redirection symbol.
    7. In C++, the standard stream stderr is used to initialze the cout object.
      stderr initializes the cerr object.
    8. The operating system stream stdin is connected to your monitor by default.
      It is connected to your keyboard.
