1. Consider this class hierarchy. Which classes use multiple inheritance?

    1. istringstream
      It inherits from istream and ios but each class has only a single parent. This is multi-level single inheritance.
    2. iostream
      Has two base classes: istream and ostream.
    3. ofstream
      Like istringstream it uses multi-level single inheritance.
    4. fstream
      It has only a single base class, but that class uses multiple inheritance, so fstream does as well.
