1. Which symbol(s) can be used to redirect the output to a file or to another program?

    1. cat < a.txt > b.txt erases the contents of b.txt before writing to it.
    2. cat < a.txt > a.txt makes a copy of a.txt
      This completely erases a.txt Never use the same file for input and output.
    3. cat < a.txt >> b.txt erases the contents of b.txt before writing to it.
      Because you are using >> for output redirection, it appens instead of overwriting.
    4. The input redirection symbol, <, asks the operating system to open a file and pass its contents to your program as standard input.
    5. You can send output from cout and output from cerr to a single file by using the redirection symbol >.
      This will only redirect standard output (cout). Error messages will still go to the screen.
