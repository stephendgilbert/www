1. Examine the following fragments of code. Indicate whether the arguments should be passed by value, reference or const reference.

    Code Argument Passing Explanation 
    int a = 3, b = 2;
swap(a, b);
    		 reference These are input-output parameters. The variables need to both be changed. 
    print("message")
    		 const reference This is a string literal, so the input parameter needs to be const reference. 
    string s = "frog";
string s2 = upper(s);
    		 const reference Even though s is not a literal, this is also an input parameter. 
    string s = "frog";
upper(s); // converts s to upper case
    		 reference This is an input-output parameter. 
    char c;
if (cin.get(c)) cout << c;
    		 reference This is an output parameter. 
    double n = 3.5;
cout << sqrt(n);
    		 value This is an input parameter of a primitive type.
