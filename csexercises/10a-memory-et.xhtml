1. Write expressions which produce the requested value. Type error if the expression cannot be written.

    Requested Value Expression Explanation
    Number of bytes used by the type long. sizeof(long) Because long is a type, you must put it in parentheses.
    Bytes used by the variable z. sizeof z With a variable, you don't need to use parentheses (although you may).
    The location where the variable y is stored. &y The address-of-operator returns the location in memory of a variable.
    The location where the value 2 + 2 is stored. error You can only take the address of a variable (lvalue) not a temporary (rvalue).
