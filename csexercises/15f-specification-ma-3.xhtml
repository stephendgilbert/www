1. Consider the following classes:

    class Pet
{
public:
   virtual void speak() const = 0;
   virtual void license() const;
   virtual int getID() const;
};
class Dog final: public Pet
{
public:
   void speak() const;
   void license() const;
};
class Cat: public Pet
{
public:
   void license() const final;
};
class Poodle : public Dog
{
public:
   ...
};
class Persian : public Cat
{
public:
   void speak() const;
};
    1. The Poodle class will not compile.
      It's base class, Dog is final.
    2. The Cat class will not compile.
      Becuase it does not override speak is is abstract, but it compiles.
    3. The Persian class is an abstract class.
      It is concrete since it overrides speak.
    4. The Dog class is a concrete class.
      It overrides speak.
    5. The Pet class is an abstract class.
      It has one abstract method.
    6. The Persian class may override license as well as speak.
      No. license is final in Cat.
