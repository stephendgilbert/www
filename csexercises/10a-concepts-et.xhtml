1. Examine the following program. Then supply the requested information. Use the following terms: block, file, static, automatic, dynamic, internal, external, none, stack, heap. 

    int a{42};
static int b{35};
void func(int c)
{
   int d = c;
   static int e{15};
   int * f = new int(3);
}
    Requested Information Answer Explanation
    Scope of the variable a file Visible from point of declaration to end of file.
    Scope of parameter c block It is only visible in the body of the function.
    Storage class for varible a static Stored when program loads.
    Storage class for varible d automatic Local variables are stored on the stack.
    Storage class for varible pointed to by f dynamic Variables created with new are stored on the heap.
    Linkage of the varible b. internal Static global variables can only be used in the file they are defined in.
    Linkage of the varible e. none Static local variables can only be used in the function they are defined in.
    Where is the varaible f stored. stack While f points to a variable on the heap, f itself is stored on the stack.
    Linkage of the function func. external Unless marked static functions all have external linkage.
