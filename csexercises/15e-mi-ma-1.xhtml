1. Consider these classes:

    class Mammal
{
public:
   Mammal() { cout << "I'm warm blooded" << endl; }
};
class FlyingAnimal
{
public:
   FlyingAnimal() { cout << "I'm flying" << endl; }
 };
class Bat: public Mammal, public FlyingAnimal
{
public:
   Bat() { cout << "I'm a bat" << endl; }
};
int main()
{
   Mammal *p = new Bat;
   Bat man;
}

    Which of the following statements is true?

    1. The Bat class definition is illegal, since it has two base classes.
      Multiple inheritance is legal in C++ (although not in Java).
    2. The line Mammal *p = new Bat is illegal.
      A Bat is-a Mammal so this is legal.
    3. The line Mammal *p = new Bat will print three lines of output.
      It will. When a Bat is constructed it calls both base class constructors.
    4. The line Bat man produces the same output as the preceeding line.
      It does.
