1. Assume the following variables have been defined:

    int i;
double n;
string s;

    Which of the following reference declarations are allowed?

    1. Allowed|Not Allowed
    2. string& sr1 = s;
      sr1 is an alias for s. If you change sr1 you change s as well.
    3. const string& sr2 = s;
      Also allowed. Because sr2 is const, it cannot be used to change s.
    4. string& sr3 = "hello";
      A regular reference must refer to an l-value, but the string literal "hello" is not an l-value.
    5. const string& sr4 = "hello";
      This is fine. A const reference may refer to an r-value.
    6. double& dr1 = i;
      While i can be used to initialize a double variable (using conversion), it cannot be used to initialize a double&.
    7. int& ir1 = i;
      Fine. ir1 refers to the int variable i.
    8. const int& ir2 = 42;
      Fine. Because ir1 is a const reference, it can refer to an r-value.
