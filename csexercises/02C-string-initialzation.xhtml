1. What is the contents of the following string objects? Do not put quotes around your answers. Use error if the initialization causes an error. Use empty if the string contains no characters.

    Initializer String Object Explanation
    string a{"rabbit"}; rabbit You can create a string using uniform initialization.
    string c = string("cow"); cow You can explicitly call the string constructor to initialize an object.
    string d(""bob""); error The initializer cannot have embedded quotes without escape sequences of raw strings.
    string e('x', 5); error There is no constructor that takes a char followed by an int
    string f = R"(Say "hi")"; Say "hi" A raw string may include embedded quotes or backslashes.
    string g(3, '?'); ??? Creates a string with 3 '?' characters.
    string h = "furnace"; furnace You can create a string using legacy initialization, just like Java.
    string i(3, "ow"); error The second argument must be a char, not a string literal.
