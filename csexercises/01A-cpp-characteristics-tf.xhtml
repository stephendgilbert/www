1. Which of these statements applies to C++?

    1. It compiles to native machine code.
      It produces native executables.
    2. Compiles to intermediate bytecode.
      While the compiler may use bytecode or assembly language during compilation, the final result is machine or native code.
    3. C++ programs are interpreted by a virtual machine (the CPPVM) when they run.
      Java programs work this way. C++ programs do not.
    4. C++ programs run directly on your CPU.
      There is no intermediate supervisor program.
    5. C++ programs are generally more efficient than Java or Python programs.
      Yes, because they don't require a supervising VM.
    6. C++ is strictly a low-level language
      C++ is a multi-paradigm langauge that has support for both low-level features (like pointers), and high-level features like classes. In general, it is considered a HLL (high-level langauge).
    7. Has features which automatically catch programming errors, like array out-of-bounds access
      The C++ language generally lacks such saftey features. It is up to you, the programmer, to supply them.
