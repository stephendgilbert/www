1. Given these classes:

    class Question { };
class ChoiceQuestion : public Question
{
public:
   void addChoice(const string& ans, bool correct);
};

    Which of the following statements are legal?

    1. Question* p = new ChoiceQuestion;
      The new expression returns a ChoiceQuestion* which you can assign to a Question* pointer.
    2. ChoiceQuestion* q = new Question;
      You cannot assign a base class pointer to a derived class pointer.
    3. Question* p = new ChoiceQuestion;
p->addChoice("Yes", true);
      Even though p points to a ChoiceQuestion, it is declared as a Question* and you cannot call the addChoice member function.
    4. ChoiceQuestion* q = new ChoiceQuestion;
q->addChoice("No", false);
      q is a ChoiceQuestion* and you can call the addChoice member function.
