1. Which of these statements are true?

    1. In assembly language, programmers use names to represent data.
      Assembly language uses the data's address.
    2. High-level languages use names to represent data.
    3. With high-level languages, variables are "chunks" of arbitrary bits.
      High-level languages organize data by its type.
    4. High-level languages provide built-in data types.
    5. In C++ the primitive types include int, double and char.
    6. The ability to define your own types mean a language is "object-oriented".
      This means the language is object-based. Object-oriented languages allow users to create a hierarchy of classes using inheritance and polymorphism.
