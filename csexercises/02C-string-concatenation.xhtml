1. Assume you have the following code:

    string s1{"peter"}, s2{"piper"}, s3{"mary"}, ans;

    The variable ans will be printed after each concatenation. What is printed? Do not put quotes around your answers. Use error if the concatenation causes an error.

    Expression Output Explanation
    ans = s1 + s2; peterpiper Concatenation does not add spaces.
    ans = "hello" + " " + "world" error You cannot concatenate two string literals (as you can in Java).
    ans = s1 + " " + s2; peter piper You may concatenate a string literal to a string object.
    ans += " picked"; peter piper picked The shorthand assignment operator appends to an existing string.
    ans = s4 + 42; error You cannot concatenate string objects and integers (as you can in Java).
    ans = s1 + ' ' + s2; peter piper You may concatenate char variables or char literals to string objects.
    ans = "So, " + s3 + " had"; So, mary had You may concatenate a string literal and a string object and aother string literal.
    ans = "So, " += s3; error The += does not work when applied to a string literal (on the left)
    ans = s3 += ", mary"; mary, mary Using += not only concatenates, but changes s3 as well. Do not do this!
