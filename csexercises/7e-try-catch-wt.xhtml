1. This walkthrough shows how an exception is caught. The stoi function throws an invalid_argument exception when it is called with a string that does not contain the digits of an integer. Assume that the program fragment reads the following input:

    42 five

    int result = 0;
int total = 0;
while (result != -1)
{
   string input;
   cin >> input;
   try
   {
      result = stoi(input);
      total = total + result;
   }
   catch (invalid_argument& ex)
   {
      result = -1;
   }
}
