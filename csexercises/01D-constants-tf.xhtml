1. Which of the following is a constant declaration that is syntactically, semantically, and stylistically correct, according to the Course Reader?

    1. const int kDaysPerWeek = 7;
      The number of days in a week is a constant.
    2. final double PI = 3.14159263589793;
      In C++ you use the reserved word const (not final) to indicate a constant.
    3. const int kDaysPerMonth = 30;
      the number of days in a month is not a constant.
    4. const int HOURS_PER_DAY = 24;
      In C++, an all caps name represents a MACRO rather than a constant. In this class we'll use the Google style for constants: kHoursPerDay.
    5. const string kWeekDays = "MoTuWeThFrSaSu";
      Constants can have any type. This is a string constant.
