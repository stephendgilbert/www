1. Given these classes:

    class Car
{ 
public: 
    void drive();
    void turn();
    int speed() const;
    void playMusic();
    ...
};
class MusicPlayer: Car
{
public:
    using Car::playMusic();
};

    Which of the following statements true?

    1. Every MusicPlayer is-a Car.
      This is private inheritance so every MusicPlayer is implemented in terms of Car.
    2. This code is correct: MusicPlayer p; p.playMusic();
      The using declaration makes the Car::playMusic member accessible in MusicPlayer.
    3. This code is correct: MusicPlayer p; p.turn();
      The MusicPlayer only inherits the implementation of Car, not the interface.
