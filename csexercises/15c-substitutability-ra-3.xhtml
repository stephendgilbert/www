1. Consider Vehicle class hierarchy shown here:

    A Road class simulates a road on which vehicles travel. Its add member function has a parameter of type Vehicle&. Rearrange the following lines to add appropriate objects to the road. You can add them in any order. Not all lines are useful.

    ^Road road1;
+Truck truck1;
+Sedan beetle;
+string bmw = "BMW 540i";
+Motorcycle harley;
+Person harry("Harry Smith");
+Trafficlight tl1;
.road1.add(truck1);
,road1.add(beetle);
,road1.add(harley);
-road1.add(bmw);
-road1.add(tl1);
-road1.add(harry);
-road1.add("beetle");
