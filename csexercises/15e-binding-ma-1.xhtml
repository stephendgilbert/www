1. Consider these classes:

    class Question
{
public:
   void setText(const string& question_text);
   void setAnswer(const string& correct_response);
   virtual bool checkAnswer(const string& response) const;
   virtual void display() const;
   ...
};

class ChoiceQuestion : public Question
{
public:
   void setText(const string& question_text);
   void addChoice(const string& choice, bool correct);
   void display() const;
   ...
};

    Which of the following statements is true?

    1. The display function uses early binding.
      Because it is virtual it uses late binding.
    2. The setText function is overridden in ChoiceQuestion.
      Because it was not virtual in Question it is not overridden, but hidden.
    3. The setAnswer function in ChoiceQuestion uses early binding.
      A function that is not virtual always uses early binding.
    4. The display function in ChoiceQuestion is virtual.
      It is not required to declare the overriding method as virtual, but it is still a virtual function.
