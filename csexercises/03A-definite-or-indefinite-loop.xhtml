1. Examine the following loops. Which are definite and which are indefinite?

    1. Definite|Indefinite
    2. for (int i{10}; i > 0; --i) {}
      Counter controlled loops are definite loops.
    3. int i{0}, n; while (cin >> n) { ++i; }
      Although this loop uses a counter, the loop is controlled by running out of data. This is an indefinite loop.
    4. int i{0}; while (i < 10) { ++i; }
      This is also a counter-controlled definite loop.
    5. for (auto i{str.size() - 1}; str.at(i) != '.'; --i) {}
      This loop uses a counter to access the characters in a string, but it is an indefinite sentinel loop.
    6. int n, counter{0}; cin >> n; while (n != 0) { counter++; n /= 2; }
      This is an indefinite limit loop.
