1. Classify the following loop fragments as counter-controlled, data, sentinel, limit or range. Use error if the loop is not one of these types.

    Loop Category Explanation
    for (auto c : str) range The range loop automatically refers to each element in str.
    do ... while (str.at(++i) != '.'); sentinel This indefinite sentinel loop stops when it encounters a period.
    for (str.at(i) != '.') ++i; error There are two versions of the for loop, but this is not one of them.
    while (cin >> str) cout << str; data This is a data loop. It ends when there is no more data.
    while (a - b > 0) { b ++; } limit This limit loop involves monitoring the difference between two variables.
