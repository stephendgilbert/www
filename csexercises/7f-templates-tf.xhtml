1. Which of these statements are true?

    1. Templates should be declared in a header and defined in a .cpp file
      Templates must be seen at compile time, so they are usually placed in a header file.
    2. It is possible to have more than one type parameter in a template.
    3. Writing template <class T> means that your template only works with class types.
      The keyword class has a special meaning when used here. For that reason, many prefer to use the alternative, typename.
    4. Templates allow you to write functions that work with a variety of types.
    5. template <typename T> and template <class T> mean exactly the same thing.
    6. In template <class T the identifier T is called the value parameter.
      The correct term is type parameter.
    7. Instantiation means that the compile generates an overloaded function from a function template.
