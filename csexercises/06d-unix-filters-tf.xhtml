1. Which command displays the names of the files in a folder in reverse order? You might need to experiment.

    1. ls -r
    2. ls | wc -l
      This counts the number of files in a directory by lines.
    3. ls | sort
      This sorts the files in a directory.
    4. ls | sort -r
    5. ls -r | sort
      This lists the file in reverse order, but then sort puts them back in sorted order.
