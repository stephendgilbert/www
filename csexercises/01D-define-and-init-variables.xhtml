1. Define and initialize the following variables.

    Variable Definition Explanation
    Define a variable price to contain the unit price of a single bottle, which is $1.95. Use an explicit type and uniform initialization. double price{1.95}; The type is double because a price can have a fractional part.
    Define a variable quantity to contain the number of bottles purchased, and direct initialize it with 6. This should work in any version of C++. int quantity(6); The type of this variable is int because one cannot purchase fractional bottles.
    The store decides to sell bottles in packs of four instead. Write a statement to adjust the value of the quantity variable. quantity = 4; Note that there is no int type before the variable name because this variable has already been defined.
    Define another variable total and initialize it with the total price from the quantity and price variables. double total{quantity * price}; Now you need to specify the type double (or auto) because this is a new variable.
