1. Which symbol(s) can be used to redirect the output to a file or to another program?

    1. <<
      This is not actually a redirection symbol.
    2. >>
      This append output to a file.
    3. <
      This is the input redirection symbol.
    4. >
      This will redirect standard output to a file in overwrite mode. is make test.
    5. |
      This will redirect standard output to another program (a pipe).
