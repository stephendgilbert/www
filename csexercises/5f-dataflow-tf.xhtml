1. Which of these statements are true?

    1. You should always use pass-by-value for class or library types
      Never use pass-by-value for these types.
    2. Input-output parameters whould be passed by reference
    3. Input parameters of type string should be passed by reference
      string input parameters should be passed by const reference.
    4. Output parameters of any type should be passed by reference.
    5. An output parameter is a value returned via the return statement.
      An output parameters value is returned via the parameter list.
    6. If a function returns more than one piece of information, you should use reference parameters.
    7. The Doxygen tag @param[out] indicates that the parameter is an output parameter.
    8. You can use the return statement from inside any function to terminate your program.
      Use the exit() function to do this.
    9. When printing error messages to the screen, you should use the cout object.
      This is what the cerr object is for.
