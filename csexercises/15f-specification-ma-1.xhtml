1. Consider these classes:

    class Shape
{
public:
   virtual void draw() const = 0;
   . . .
};

class Circle : public Shape
{
public:
   void draw() const;
   . . .
};

class Rectangle: public Shape
{
public:
};

    Which of the following statements is true?

    1. Shape is an abstract base class.
      Any class with a pure-virtual function is an ABC.
    2. The line Shape s; is legal.
      You cannot instantiate (create an instance of) an abstract class.
    3. The line Shape *p; is legal.
      Although you cannot create instances of abstract classes, you may create pointers to them.
    4. Rectangle is an ABC
      Because it does not override draw() it is also an abstract class.
