    NumericQuestion Function extend (e), replace (r), inherit (i), new (n) Explanation
    setText inherit This function is not defined in the derived class. The setText function of the Question base class is inherited.
    display extend This function overrides the display function of the base class, and it extends it. First, the base class function is called, and then something else happens.
