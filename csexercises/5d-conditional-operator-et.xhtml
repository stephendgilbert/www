1. Assume that you have the following variable:

    int a, b, c;

    Use the conditional operator to write an expression that carries out the requested operation.

    Operation Expression Explanation
    Assign 2 to a if b is greater than c and 1 otherwise. a = b > c ? 2 : 1; The true value follows the ?, the false follows the :
    Assign 3 to a if b is greater than c and leave a unchanged otherwise. a = b > c ? 2 : a; The variable a is assigned to itself if b is not greater than c.
    Assign 4 to a if b is greater than c, 5 to a if b is less than c and leave a unchanged otherwise. a = b > c ? 4 : b < c ? 5 : a; The false condition uses another conditional expression.
