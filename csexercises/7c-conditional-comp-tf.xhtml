1. Which of these statements are true?

    1. When you link your program, you may have several copies of some functions
      If you do, you will get a clash of symbols. Your executable can have only definition for each function.
    2. Use the preprocessor directives #if and #endif to conditionally compile a portion of your code.
    3. #if conditions can refer to any integer variable
      You may not use variables at all with #if. You can only use #defined constants.
    4. You can use the preprocessor function defined() to see if a preprocessor constant exists.
    5. The statement: #if defined(A) is evaluated when your program runs.
      This is evaluated at compile time, not at runtime.
    6. You can use the predefined preprocessor constant __cplusplus to check which version of C++ was used to compile your program.
