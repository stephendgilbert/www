Week 1—August 25 to August 31
Introductions

Welcome to CS 170. Before you learn how to program in Java you'll have to spend some time getting prepared. In this first unit, you'll cover some background material, introduce yourself to your fellow classmates, and set up your Java development environment.

This week you'll learn:

  • how your computer's memory stores data
  • how your computer's CPU converts data to information
  • a brief history of computer programming
  • what Java is, and how it works
  • how to set up your Java programming environment

To the right are links to the slides from the first in-class lecture as well as the online course materials for this unit. Along with the lessons here, you should also read the first part of Chapter 1 in your textbook. Next week we'll look at writing Java programs, finishing up Chapter 1.

In addition to the reading, you'll need to complete the lab and homework exercises and submit them by noon on Tuesday, September 2. (Monday, September 1 is Labor Day and so there will be no class then.)

By Tuesday you should:

  • listen to each of the online lessons.
  • complete and submit the Week 1 homework assignments.

See you online!
--Steve
