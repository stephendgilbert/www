Week 8— October 13 to October 19
Making Decisions

Welcome to CS 170 Week 8. From 2001's HAL to War Games' WOPPER to Speilberg's AI to Arnold's Terminator, popular culture has often portrayed computers as "thinking machines" rather than glorified calculators.

Computers seem to "think" because, unlike an adding machine, a computer can make decisions and carry out different actions, based upon the input it receives.

This week you'll teach your computer how to make decisions, by learning how to:

  • use relational operators to compare object types
  • perform Boolean operations using the logical operators
  • calculate multi-way branches using sequential and nested if statements
  • write code that employs Java's switch statement
  • craft Boolean expressions using Java's selection operator
  • simplify Boolean conditions by using DeMorgan's law

You'll also improve your "nigh-watchman" robot by adding constructors and writing overloaded, decision-making methods.
