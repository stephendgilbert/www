Week 3—September 8 to September 14
Objects and Variables

Welcome to CS 170 Week 3. Java is an object-oriented programming (OOP) language, and this week you'll start using the objects that are part of Java. You'll start exploring OOP concepts using the BlueJ Integrated Development Environment (IDE) and then, you'll learn how to create objects and tell them what to do.

This week you'll learn:

  • how to create simple variables and assign values
  • how to create object variables and construct objects
  • how to pass parameters to customize your objects
  • how to read and navigate the Java documentation
  • how to create and constrol your own online robot

To the right are links to the handouts from this week's in-class lecture as well as the online course materials for this week. Along with the lessons here, you should also read Chapter 2 in your textbook (up to the Graphics material). We'll have a quiz on Chapter 2 during the on-campus class next week.

In addition to the reading, you'll need to complete the lab and homework exercises and submit them by Noon on Monday, September 15.

By then you should:

  • listen to each of the online lessons. (Most lessons this week include hands-on lab exercises, so you should budget at least twice the running time of the lecture to complete them.)
  • complete and submit the ic04 online lab exercises.
  • complete and submit the Week 3 homework assignments.

See you online!
--Steve
