Week 1

Introduction to CS 170

August 25-31, 2008

Welcome to Week 1: Introduction to CS 170. Before you learn how to program in Java you'll have to spend some time getting prepared. In this first unit, you'll cover some background material, set up your Java development environment, write and run your first Java application, and get aquainted with the basic structure of a Java program.

  • how your computer's memory stores data
  • how your computer's CPU converts data to information
  • what Java is, and how it works
  • how to set up your Java programming environment
  • the mechanics of Java programming
  • the basic syntax of a Java program

By next Sunday, you'll need to purchase your textbook and read Chapter 1. In addition, you'll need to complete and submit your first homework assignment as well as two in-class exercises, one for the in-person lecture, and one for the online portion of the lecture. You'll find information on each of these on the separate homework and in-class exercise pages available from links on the syllabus.

Welcome aboard, and good luck.

Lectures

This semester, CS 170 is a hybrid class; we'll have 11 in-person lectures and 13 online lecutures. Rather than a single monolithic online lecture each week, though, I've arranged the class as several mini-lectures so you won't have to set aside several hours for a single sitting. You can listen to each lecture by clicking the hyperlinks below. I've organized the material sequentially so while you can skip around, you'll probably find it easier to take the material in order.

The online lectures themselves are Adobe Flash presentations, so you should be able to view them in most any Web browser. If you need to use the computers at school or in the library, though, remember to bring your own headphones or earbuds; trying to listen to the lecture by reading the captioning at the bottom of each slide is pretty difficult.

I've also provided PDF "handout-style" copies of the slides used in the in-person lectures. You can print these out before coming to lecture, and use them to take your own notes.

Get Started with C170: Lecture 1

In our first class meeting, we'll start out by going through the course syllabus together. The syllabus is kind of like a contract; it explains what you can expect to learn in the class, and, it tells you what will be expected from you in terms of effort, performance, and behavior.

The next stop is the CS 170 online classroom classroom tour. I'll introduce you to the main features of the online lectures, show you how to submit your assignments, and make sure you get signed up for the class discussion group.

Finally, we'll spend some time going over some fundamental computer concepts that you've probably covered in earlier computing classes: how memory and the CPU work and how you use the command line to "talk" to your computer.

Meet the Gang

After you've looked around and made yourself at home, I'm sure you'll want to meet all of the other denizens of Casa CS 170, and I know that they're all anxious to find out all about you. To do that, use the class discussion board to introduce yourself to the rest of the class.

Getting Started with Java: Lecture 2 (Online)

Do you have what it takes to be a successful online student? Of course you do! Being a successful student, (online or otherwise) isn't an inherited trait, like your height or eye color. Instead, its a matter of learning a few simple habits and then applying them. (This is the only lecture this week that isn't specifically targeted to CS 170). (22 min.)

Next week we'll review some of the basic HTML coding that you covered in CIS 123, and get started with CSS. Before we do that, though, you'll need to spend a little time getting your computer set up with the software tools you'll need to complete the course.

All of the software we'll use for this class is free, so you won't have to spend any money. I've also selected programs that are available for Windows, the Mac, and Linux, so you can work on the platform of your choice.

Most of you will have some of these programs installed already, but these lectures go step-by-step for those who are starting from scratch. While you should feel free to skim over the instructions for those packages you already have working, make sure you do at least browse through the lecture so you don't miss any CIS-223-specific instructions.

Here's a list of the software we'll use this semester:

  • The Firefox Web browser along with several Web development add-ons.
  • A plain-text programmer's editor. For Windows or Linux, I recommend the Scintilla Text Editor. You Mac folk should use TextWrangler2 from BBEdit software.
  • A word processor that can include graphics and that can export files in Portable Document Format (PDF). (This lecture also has the information you'll need to prepare and submit your homework assignments.
  • A way to easily capture screenshots as you work through the weekly lectures.(This lecture also has the info on starting your first homework assignment.)

Finishing Up
