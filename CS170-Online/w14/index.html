Week 14— November 24 to November 30
Widgets and Inheritance

Welcome to CS 170 Week 14. This week and next week we'll be looking at selected topics in Chapters 9, 10 and 11 in your textbook. You won't be responsible for all of the material in these three chapters; just what we cover in lecture. This week we'll start working with Java's graphical user interface (GUI) widgets and you'll be introduced to event handling.

This week you'll learn:

  • how to write GUI programs that contain JButton and JLabel widgets
  • how to create a new class using inheritance
  • how to identify, create and use an abstract class
  • how to write and override an abstract method
  • how to define and implement a Java interface
  • how to implement Java event handling

To the right is a links to the slides from the in-class lecture this week. There are no on-line lectures because of the Thanksgiving holiday. There are also no homework exercises due next week. You should also skim through Chapters 9 and 10 in your textbook to reinforce the material from the in-class lecture.

See you next week in class!
