Week 4—September 15 to September 21
Implementing Classes

Welcome to CS 170 Week 4. This week, we'll shift our focus from using objects to defining classes and methods. In your text, you'll learn how to define simple standalone classes, such as BankAccount and CashRegister that you can use just like you use the "built-in" String or Rectangle classes. You'll also be introduced to Java applets in our first graphics interlude.

This week you'll learn:

  • the basic structure of a stand-alone class
  • how to write mutator and accessor methods
  • how to write constructors that intialize your object
  • how to write test programs manually and using JUnit
  • how to create a simple Java applet

To the right are links to the handouts from this week's in-class lecture as well as the online course materials for this week. Along with the lessons here, you should also read Chapter 3 in your textbook (up to Section 3.9). We'll have a quiz on Chapter 3 during the on-campus class next week.

In addition to the reading, you'll need to complete the lab and homework exercises and submit them by Noon on Monday, September 22.

By then you should:

  • listen to each of the online lessons.
  • complete and submit the ic06 online lab exercises.
  • complete and submit the Week 4 homework assignments.

The lessons this week include extensive hands-on lab exercises and the homework assignments require a lot of programming. You should budget at least twice the running time of the lectures this week, possibly more, to complete them. Make sure you leave time for the homework assignments as well.

See you online!
--Steve
