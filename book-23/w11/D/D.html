Arrays and Loops

Just as with vector, the real advantage of arrays is that you can automate the processing of a collection of related elements, like the grades for all the students in a class. However, because arrays are lower-level structures, processing them is not quite as convenient.

Most commonly, you'll use a counter-controlled loop, like this loop, which could be used to compute the average of the elements in an array: 

int count = 0;
double sum = 0;
for (size_t i = 0; i < kSize; ++i) {
    sum += array[i];
    count++;
}

Unlike string and vector, arrays have no built-in size() member function. That means, for a counter-controlled loop, the array's length must be stored in a variable prior to entering the loop. Since the array's length is a constant, you'll normally use the same constant used to allocate the array.

Other Loops

You may use the range-based for loop on arrays, provided that the array definition is still in scope. Here's an example: 

int a[] = {...};
for (auto e : a)
{
   cout << e << " ";
}

Similarly, you can use an iterator loop using pointers, like the loops you used to process images in the previous lesson. Finally, you can use a sentinel loop provided you place a special value in the array to mark its end. This is used with character arrays, as you'll see when we cover C-style strings.
