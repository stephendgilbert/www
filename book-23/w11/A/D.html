The Darken Filter

The first filter we'll write is "darken". Here's the algorithm: 

For every byte 
  if it is greater than 64 then
    subtract 64 from it
    rewrite the byte

Here's what the processing code looks like: 

while (beg != end)
{
    if (*beg > 64) {
        *beg = *beg - 64;
    }
    beg++;
}
