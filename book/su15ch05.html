CS 170 Introduction to Java Programming

Procedures and Functions


Welcome to Chapter 5. In the last few chapters, you've put all of your code in a single method such as main(), run() or paint(). In this chapter, you'll learn how to write programs with multiple methods that work together to carry out a goal.

We'll start by learning how to define independent procedures; subprograms which which carry out an action. Then, you'll move onto creating functions, which take some input (as parameters) and then calculate and return a value.

In this chapter you'll learn how to:

  • write programs with multiple procedures
  • write user-defined procedures that take parameters
  • write user-defined functions that return values.
  • understand the style rules you should follow
  • recognize and prevent runtime errors in your code
  • use the DrJava debugger to examine and suspend your programs as they run.
