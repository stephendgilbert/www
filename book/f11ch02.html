CS 170 Introduction to Java Programming

Data and Output


Welcome to Chapter 2. The last chapter was mostly preparation and memorization; in this chapter, you'll start learning what those mysterious incantations you typed into DrJava actually mean!.

The focus of this chapter is on output. You'll learn how to display textual data, and start working with some of Java's numeric and object types. We'll finish up this chapter by returning to your Java applet and learning how to display graphical output.

In this chapter you'll learn:

  • how to use Java's console output objects
  • how to create ACM console applications
  • how to create numeric variables and fill them with values
  • how to use Java's different numeric types
  • how to write graphical output statements using the Graphics class
