CS 170 Introduction to Java Programming

Objects, Input and Processing


Welcome to Chapter 3. In the last two chapters, you learned how to perform output and create variables. In this chapter you'll learn how to write programs that get input and that perform computations. After all, that's why they're called computers, right?.

In this chapter you'll learn:

  • how to create objects and send them messages
  • how to retrieve input from your program's users
  • how to perform calculations to process the input
  • how to work with Math functions
  • how to create Color and Font objects
