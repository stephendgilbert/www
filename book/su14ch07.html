Introducing Loops


Welcome to Chapter 7 This chapter continues the application of computer logic as you learn how to make your programs repeat a set of actions, using two different kinds of loops. In addition, you'll develop a strategy for writing correct loops the first time.

In this chapter week you'll learn:

  • use the for loop to generate data
  • a strategy for developing correct loops.
  • to write programs using the while loop.
  • to write loops controlled by a condition.
  • to generate different sequences of random numbers.

Learning about Loops

7.1—Loop Concepts
Learn how loops allow you to write code that repeats a set of actions.

7.2—The for Loop
Learn how to write counter-controlled loops, using Java's for statement.

7.3—Counted while Loops
Learn how to use the while statement to write loops use a counter.

7.4—WritingLoops
Learn about a loop's bound and goal and develop a strategy for writing loops that are correct.

7.5—Sentinel Loops
Introducing indefinite loops using the while statement and a sentinel to end the loop.

7.6—Random Numbers
Generate random numbers with the Random and ACM RandomGenerator classes.
