CS 170 Introduction to Java Programming

Introducing GUI Apps


Welcome to Chapter 8. In this chapter you'll learn how to create graphical ACM programs, and how to work with different kinds of graphical objects. You'll also learn a little bit about using the graphical widgets provided by Java's GUI package called Swing.

Before we get to that, let's talk about the midterms and makeup exams that will take place on Thursday, July 3.

Midterm Exam

The Midterm Exam will be on Thursday, July 3, starting at 8:00 AM. The exam will have 100 questions, and you'll have 2 hours to complete it. You'll take it in class on the computer using the SecureExam browser. You can get an idea of the kinds of questions you'll be asked by looking at your past reading quizzes.

After the Midterm Exam, you will be given the chance re-try one (or two) of the first five Proficiency Exams. If you get a better score on the retake, it will replace your original score. Graphics Track

8.1—ACM Graphics Programs
Learn how to write object-oriented graphical applications, using the classes in the ACM graphics package.

8.2—Meet the GObject
Learn how to use the methods in the GObject class to control the graphical objects you create.

8.3—Fillable and Scalable
Learn how to use the interfaces in the GObject class to fine-tune your objects.

8.4—GImage and Friends
Learn how to use the ACM GImage class along with several others, to create sophisticated ACM applications.

8.5—Meet the Widgets
Learn how to use Java's Swing wigets, starting with the JLabel class.
