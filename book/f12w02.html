CS 170 Introduction to Java Programming

Data and Output


Welcome to Chapter 2. In the last chapter, you learned how to display simple text output, and were introduced to the System.out object, the String class, as well as the Applet and Graphics classes used for graphical output.

In this chapter, we'll begin by covering a little background material, that we skipped over last week, so you could dive write into writing your own programs. After that, you'll begin looking at different data types, including strings and numbers. We'll finish up this chapter by learning about one more kind of Java program, the ACM ConsoleProgram.

In this chapter you'll learn:

  • about the history of computer programming
  • what Java is, and how it works
  • how to work with Java's String type
  • how to create numeric variables and fill them with values
  • how to use Java's different numeric types
  • how to write use the external ACM libraries to write more capable console programs.
