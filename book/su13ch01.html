CS 170 Introduction to Java Programming

Writing Simple Programs


Welcome to CS 170. Before you learn how to program in Java you'll have to spend some time getting prepared. In this first chapter, you'll cover some background material, set up your Java development environment, and write and run your first Java programs.

In this chapter you'll learn:

  • how your computer's memory stores data.
  • how your computer's CPU converts data to information and instructions.
  • the mechanics of Java programming.
  • the basic syntax of a Java console program.
  • how to use Java's console output objects.
  • how to print text output using the String type.

To the right you'll find the links to the individual sections in this chapter. On the exercises page you'll find links to those exercises and homework assignments that accompany this chapter.

Remember that during this compressed Summer session, the homework assignments are entirely for your benefit. You can check your work by using the Check button in DrJava, but you won't submit any of the assignments, and they will not contribute to your course grade.

Note: If you want to follow along on your own computer with the programming lessons, (highly recommended), then you must install and configure your Java programming tools first. Your instructor will provide you with a handout telling you how.
