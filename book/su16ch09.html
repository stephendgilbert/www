CS 170 Introduction to Java Programming

Loops, Events & Animation


Welcome to Chapter 9 In this chapter you'll continue working with loops, learning how to process Strings and files.

You'll also continue working with Java's graphical widgets, and learn how to respond to different kinds of events, so you can write interactive GUI programs.

Then, you'll put both loops and widgets together to start writing animated programs like video games.

In this chapter you'll learn:

  • how to write indefinite loops that test a condition.
  • how to use Java's jump statements
  • how to use iterators to process Strings and files
  • how to animate your graphical applications
  • how to use Java's event handling system to create programs that respond to mouse clicks.
  • how to use Java's widgets to make your GUI apps interactive.
