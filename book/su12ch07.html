CS 170 Introduction to Java Programming

More on Logic


Welcome to Chapter 7 This chapter continues the application of computer logic as you learn how to make your programs repeat a set of actions, using two different kinds of loops. In addition, you'll develop a strategy for writing correct loops the first time.

Finally, you'll work with several of the more sophisticated objects defined in the ACM graphics package.

In this chapter week you'll learn:

  • use the for loop to generate data
  • a strategy for developing correct loops.
  • to write programs using the while loop.
  • how to how work with ACM images and shapes.

To the right you'll find the links to the individual sections in this chapter. During this abbreviated Summer Session, you'll find links to those exercises assigned by your instructor in BlackBoard.

Learning about Loops

7.1—The for Loop
Learn how to use Java's for statement to create loops that use a counter to control repetition.

7.2—WritingLoops
Learn about a loop's bound and goal and develop a strategy for writing loops that are correct.

7.3—The while Loop
Learn how to use the while statement to write loops use a counter and those that test a condition.

7.4—Loop Functions
Hands-on practice writing functions that use the for loop and the while loop.

Graphics Track

7.5—ACM Graphics
Learn how to write object-oriented graphical applications, using the classes in the ACM graphics package.

7.6—GImage and Friends
Learn how to use the ACM GImage class along with several others, to create sophisticated ACM applications.
