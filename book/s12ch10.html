More on Arrays


Welcome to Chapter 10. In this chapter you'll continue your work with arrays, learning how to create and process object and two-dimensional (2D) arrays.

You'll also learn how to work with command-line arguments and use variable-length parameter lists. Then, we'll put some of those concepts to work, using 2D arrays to process your digital photos.

Finally, we'll end this chapter by learning how to add use arrays with animation and how to respond to keyboard events.

In this chapter you'll learn:

  • how to create and use object and 2-D arrays.
  • how to process an application's command line.
  • how to use variable length parameter lists.
  • how to apply arrays to image processing.
  • how to animate arrays and process keystrokes.
