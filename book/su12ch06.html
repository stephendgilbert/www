Introducing Computer Logic


Welcome to Chapter 6. From 2001's HAL to War Games' WOPPER to Spielberg's AI, popular culture has often portrayed computers as "thinking machines", rather than as glorified calculators.

Computers seem to "think" because, unlike an adding machine, a computer can make decisions and carry out different actions, or repeat a set of actions, based upon the input it receives.

In this first chapter on Computer Logic, you'll teach your computer how to make simple decisions, how to combine decisions with Java's logical operators and how to handle more complex decisions. This week you'll learn how to:

  • use the boolean primitive type
  • compare numbers and Strings.
  • perform simple decisions using the if statement.
  • to combine Boolean conditions.
  • to write predicate functions.
  • to implement multi-way decisions.
  • write functions that use selection and loops.
