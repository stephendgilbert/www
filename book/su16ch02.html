CS 170 Introduction to Java Programming

Data and Output


Welcome to Chapter 2. In the last chapter, you learned how to display simple text output, and were introduced to the System.out object and the String class, as well as the concept of escape sequences.

In this chapter, we'll begin by covering a little background material, that we skipped over last week, so you could dive write into writing your own programs. You'll first learn about the the Applet and Graphics classes used for graphical output. After that, you'll begin looking at different data types, including strings and numbers. We'll finish up this chapter by learning how to write interactive programs that process user input.

In this chapter you'll learn:

  • about the history of computer programming
  • how to use Java's Graphics class for output.
  • what Java is, and how it works
  • how to work with Java's String type
  • how to create numeric variables and fill them with values
  • how to use Java's different numeric types
  • how to retrieve input from your program's users
