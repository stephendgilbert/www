CS 170 Introduction to Java Programming

Arrays & More Widgets


Welcome to Chapter 9 In this chapter you'll learn how to work with collections of values: groups of related objects that can be treated as single unit, or that can be processed with loops to automate various kinds of calculations.

We'll start with the simplest kind of collection, the Java array. We'll also make a return to the widget world, and learn how to use buttons and text fields.

In this chapter you'll learn:

  • what array variables are, and why they're useful.
  • how to define arrays and use arrays with methods.
  • how to use arrays with loops.
  • how to copy and compare two arrays.
  • how to use buttons and text-fields and how to respond to action events.
