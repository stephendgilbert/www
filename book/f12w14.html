CS 170 Introduction to Java Programming

Inheritance and Composition


Welcome to Chapter 14. In this chapter you'll learn two different techniques for creating new classes: inheritance and composition. You'll also learn about enumerated types and how to work with Java's ArrayList collection class.

In this chapter you'll learn:

  • how to create new classes from existing classes
  • how to use inheritance to extend the ACM and JFC components
  • how to create abstract classes to require behavior.
  • how to combine classes to create new types
  • how to create new single-value user-defined types
  • how to use Java's ArrayList class

Inheritance Basics

14.1—Extending Classes
Learn how to create use inheritance to create new classes by extending existing classes.

14.2—Overriding Methods
Learn how to override subclass methods, write new constructors for your classes, and how to prevent inheritance altogether.

14.3—Inheritance at Work
Learn how to use inheritance to create new versions of the ACM and JFC component classes.

Composition and Enumerations

14.4—Composite Objects
Learn how to use the technique of composition to combine existing classes to create new versions.

14.5—Enumerated Types
Create user-defined single-value types and use them to create your own PlayingCard class.

Arrays and Generic Classes

14.6—ArrayLists
Use encapsulation to hide the details of processing partially-filled arrays, and learn how to use Java's generic ArrayList class to store any type of element.
