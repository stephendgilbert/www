CS 170—Chapter 2

Homework and Exercises


In this chapter you'll concentrate on simple output programs, using Strings and escape sequences. In class you'll write "regular" console programs and start working with input. Here's the list of things you should do this week.

  1. Difference: Another console-mode program. The purpose of this exercise is to get more practice with printing escape sequences.
  2. Greeter: This is our first program with input. Much like the first console output programs you wrote, it displays a simple message. The difference it, this time when you ask the user for their name, you actually listen!

    Before you can complete this assignment, you'll need to understand the material in Section 2.6—Interactive Programs.


  3. Your First Java Applet: Building a graphical program program that can be displayed in a brower. I've provided you with the code you need to type. The purpose of this exercise is to get more practice with the mechanical programming process.
