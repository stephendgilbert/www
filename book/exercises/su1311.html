CS 170—Chapter 11

Exercises and Assignments


In this chapter you continued working with arrays, concentrating on using loops to process all of the elements in an array. These assignments will give you some practice doing these things, in anticipation of Proficiency Quiz 8—Arrays and Loops

Here's the list of things you should do:

  1. Centered Average: Make sure you know how to process arrays with loops. You should know how to count, sum, average and find the extreme values. This assignment will walk you through writing a function that does all of these things.

  2. Arrays and Loops: After you've completed Homework 36, try these two functions, arrayCount9(), and array667 to practice for the next PQ exam.

  3. More Array Algorithms: Try your hand at two more difficult array-processing functions, the sum13() and the has22() problems.

  4. 2D and Object Arrays—TicTacToe: In this chapter's code folder, you'll find starter code for a TicTacToe ACM program. Much of the program is written. It compiles and you can run it.

    You'll create a pair of 2D arrays (one JLabel and one char), and then complete several methods that process the arrays to complete the program. Inside the program, you'll find six areas you should complete, along with instructions.
