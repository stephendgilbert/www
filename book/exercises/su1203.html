CS 170—Week 3

Exercises and Assignments


The big ideas this week are using the if statement to make decisions, using the logical operators to combine decisions, and writing decisions that have more than one output. You'll use those two when writing your own functions.

This week you also learned how to repeat a series of actions, using both the for loop and the while loop. These extra exercises will give you some more practice writing functions that use these skills. Here's the list of things you should do this week.

    1. Writing Simple Logic Functions: LogicProblems: As you did last week with the String functions, you should try to complete the remaining String functions in LogicProblems.java. To get started, walk through the selection section at the beginning of Section 6.5—Flow-of-Control Functions. That will get you started with the first exercise. Once you're finished, complete the problems:
      • sleepIn (Solution)
      • sumDouble
      • diff21 (Solution)
      • makes10
      • nearHundred (Solution)

      Even if you already know how to use nested if statements, for this assignment, try to solve these using only simple if statements along with the return statement. Don't use any of the binary Boolean operators like && or ||. You may use the Boolean not operator (!), even though we haven't covered it yet.

      Just like last week, you can work on these problems without using DrJava by working with the online homework site CodingBat.com where you'll find them in the Warmup-1 section. That's what the solution videos here use. Remember, though, if you do use CodingBat instead of DrJava and Check, you cannot use the static modifier on your functions. With DrJava and Check, you may (so your code will look like that shown in your text), but you don't need to.

    2. Combining Decisions—LogicalOperators: You can get some extra practice with the logical operators by completing these predicate functions in LogicalOperators.java. To get started, walk through the hands-on exercise in Section 6.6—Hands-On Selection. That will get you started with the first exercise. Once you're finished, you can continue with these problems:
      • in1To10
      • redTicket(Solution)
      • sortaSum
      • withoutDoubles(Solution)
      • teaParty

      Complete these problems using if statements along with the binary Boolean operators like && and ||, along with the not operator (!).

      You can check your answers with the Check program.

    3. Multi-way Branches—MultipleChoice: This is the third set of decision functions this week. Like the other examples in Section 6.6—Hands-On Selection, these require you to write functions using sequential or nested if statements. Do not use the logical operators.
      • evenlySpaced(Nested Solution, Solution using &&)
      • luckySum
      • loneSum
      • makeBricks(Solution)
      • makeChocolate

      You can check your answers with the Check program.

      PQ05 - Logic Functions will be similar to these problems. You will have one-half hour to complete a similar program. If you want more practice, try the other CodingBat problems in the Logic sections.

    4. String Functions With Loops: LoopProblems: This is the second set of functions this week. Like the first two examples in Section 7.4—Loop Functions, these require you to write String functions using the for loop to solve each problem.
      • stringTimes(Solution)
      • stringE
      • catDog(Solution)
      • countCode
      • bobThere(Solution)

      If you work at CodingBat.com you'll find these problems in the String-2 section. If you use DrJava, then you'll find them in the LoopProblems.java file in this chapter's code folder. You can check your answers with Check.

    5. The while Loop—WhileLoopsProblems: This set of functions this week all involve more complex processing. Like the problem at the end of Section 7.4—Loop Functions, these require you to write functions using the while loop, even though many of them could be solved easier using the for loop.
      • repeatSeparator(Solution)
      • xyzMiddle
      • sameStarChar(Solution)
      • starOut
      • wordEnds

      You can check your answers with the Check program.

      PQ06 - String Functions using Counted Loops will be similar to these problems. You will have one-half hour to complete a similar program. PQ06 will take place on the Thursday after the Midterm Exam on July 19th.
