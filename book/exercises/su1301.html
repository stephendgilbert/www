CS 170—Chapter 1

Homework and Exercises


The exercises in this chapter are pretty simple, since we're just getting started, and because there's a lot of things you need to do to get set up. Here's the list of things you should do:

  1. Set Up Your Tools: If you're going to work at home, then follow the instructions in that handout I've provided to install Java along with the DrJava Integrated Development Environment. You'll find the link to this handout in the syllabus. If you are going to do all of your work in the Computing Center, then you only need to do this if you are going to work at home.
  2. Your First Console Program: This the exercise that goes along with the Java Mechanics lesson in your textbook.
  3. Your First GUI Program: Building a graphical program. Again, I've provided you with the code you need to type. The purpose of this exercise is to get more practice with the mechanical programming process.
  4. Printing a Message: This is the exercise that goes along with the Introduction to Strings lesson in your textbook. You'll be asked to print a program containing quotes and backslashes.
