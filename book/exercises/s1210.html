CS 170—Week 10

Exercises & Assignments


This week you'll be writing problems that use arrays and loops. These problems pick up from those you did last week, using more complex loops, processing images, and using 2D arrays of objects. Here's the list of things you should do this week.

  1. More Arrays—ArrayLoopProblems2: Here are some more array loop problems that will help you practice for the programming proficiency quiz. You'll find these in ArrayLoopProblems2.java, which you started in the text. These are a little more complex than the set from last week, but you should be able to solve them with a single loop.
    • bigDiff
    • sum13
    • has22
    • isEverywhere
    • matchUp

    You can check your answers with the Check program. Look at the solutions if you get stuck.

  2. 2D Array Processing—ImageManipulation: Complete the ImageManipulation program from the last section of Section 10.5—Image Processing. The text provides you with complete code for several of the transformations. I want you to complete the unfinished sections on your own.

    There is no Check program for this assignment.

  3. 2D and Object Arrays—TicTacToe: In this chapter's code folder, you'll find starter code for a TicTacToe ACM program. Much of the program is written. It compiles and you can run it.

    You'll create a pair of 2D arrays (one JLabel and one char), and then complete several methods that process the arrays to complete the program. Inside the program, you'll find six areas you should complete, along with instructions.

    Use the discussion board to ask any additional questions. There is no Check program for this exercise.
