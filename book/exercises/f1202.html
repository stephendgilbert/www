CS 170—Week 2

Homework and Exercises


This week you'll concentrate on simple output programs, using Strings and escape sequences. In class you'll write "regular" console programs and some ACM console programs. Here's the list of things you should do this week.

  1. H05 - Printing a Message: This is the exercise that goes along with the Introduction to Strings lesson in your textbook. You'll be asked to print a program containing quotes and backslashes.
  2. H06 - Difference: Another console-mode program. The purpose of this exercise is to get more practice with printing escape sequences.

Because of the Labor Day holiday, there are only a couple assignments this week. Make sure you have read Section 2.3 - Introduction to Strings, and have these two assignments finished by the beginning of class on Wednesday or Thursday.
