CS 170—Week 6

Exercises and Assignments


This week you'll be writing problems that use arrays and loops. These problems pick up from those you did last week, using more complex loops, processing images, and using 2D arrays of objects. You'll also start writing programs using your own user-defined types (classes). Here's the list of things you should do this week.

  1. More Arrays—ArrayLoopProblems2: Here are some more array loop problems that will help you practice for the programming proficiency quiz. You'll find these in ArrayLoopProblems2.java, which you started in the text. These are a little more complex than the set from last week, but you should be able to solve them with a single loop.
    • bigDiff (Solution)
    • sum13
    • has22 (Solution)
    • isEverywhere
    • matchUp

    You can check your answers with the Check program.

  2. 2D Array Processing—ImageManipulation: Complete the ImageManipulation program from the last section of Section 10.5—Image Processing. The text provides you with complete code for several of the transformations. I want you to complete the unfinished sections on your own.

    There is no Check program for this assignment. Click HERE to see a solution to a portion of this problem.

  3. A Simple Class—PongScore: In your textbook, you completed the PongBall class for a game of Pong. Follow those same steps on your own, and complete the classes, PongScore.

    Check your work with the Check program. Click HERE to see a solution to this problem.

  4. A More Complex Class—PongPaddle: In your textbook, you completed the PongBall class for a game of Pong. Follow those same steps on your own, and complete the classes, PongPaddle.

    Check your work with the Check program. Click HERE to see a solution to this problem.

    EXTRA. Here's a video that shows you how to test the PongPaddle class. That should help you with your assignments this week. Click HERE to see this solution.

  5. A Class from Scratch—MyEmployee: This week you learned how to create classes. Try it out. Implement the class MyEmployee. An employee has a name (a string) and a salary (a double). Provide the following constructor and methods:

    public MyEmployee(String name, double initialSalary)
    public String getName()
    public double getSalary()
    public void raiseSalary(double byPercent)

    These methods return the name and salary, and raise the employee's salary by a certain percentage. Here's a sample usage:

    MyEmployee steve = new MyEmployee("Gilbert, Steve", 95000.0);
    steve.raiseSalary(10); // gimme a 10% raise

    Check your work with the Check program. Click HERE to see a solution to this problem.
