CS 170—Week 7

Exercises and Assignments


The big idea this week is writing functions that repeat actions. You should know how to solve problems using counter-controlled loops based on both the for and while statements.

Here's the list of things you should do this week.

  1. H20—Using the for Loop: Read Section 7.1—Loop Concepts, and Section 7.2—The for Loop where you'll learn how to write functions that repeat their actions by using a counter-controlled loop. Then, complete the function doubleChar() following the instructions in the handout.
  2. H21—Counting Greetings: In this assignment, you'll use the for loop to find and count all of the instances of the word "hi" in an input String. In addition to the for loop, you'll need to add some decisions with an if statements, as well as calling methods to compare strings and extract substrings.
  3. H22—Mixing Strings: In Section 7.3—Counted while Loops and Section 7.4—Writing Loops you'll learn how to use while in place of for to write a counter-controlled loop. More importantly, you'll develop a strategy for solving loop problems. Once you've read both of those, put your skills to work solving a problem that combines two strings into one.
