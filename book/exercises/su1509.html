CS 170—Chapter 9

Exercises and Assignments


In this chapter you continue working with the application of loops. You'll learn about different kinds of indefinite loops: sentinel loops, flag-controlled loops, limit loops, and data-bound or EOF loops. You'll also learn how to use different kinds of jump statements. For the exercises in this chapter, you'll write a function using jumps, and a program that uses iterators.

Here's the list of things you should do:

  1. Using Java's Jump Statements—Summing Digits: Read Section 9.1—Sentinel Loops, and Section 9.2—Java's Jumps where you'll learn how to write programs using the break and continue statements. Then, put what you've learned to work writing yet another version of the sumDigits() function; this time, one that sums the digits in a String.
  2. Using Iterators—File Stats: Read Section 9.4—Iterators and Loops to learn how to use the Scanner class to process the data in a text file. Then, complete the FileStats console-mode program to open and read a file, and count the number of lines, words and characters that it contains.
  3. Complex Loops—Same Ends: Get more practice, honing your loop-building skills, and trying out some more complex problems. For this assignment, write the function sameEnds which finds the largest, non-overlapping substring appearing at both ends of the given String.
  4. Complex Loops—Max Block: Even more loop-building and logic practice. For this assignment, write the function maxBlock which finds the length of the largest "run" in the given String. A run is a sequence of adjacent characters.
