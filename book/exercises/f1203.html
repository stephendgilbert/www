CS 170—Week 3

Homework and Exercises


You'll begin this week by taking Proficiency Quiz 1—Console Output. Then, you'll begin moving into writing interactive programs. You'll concentrate this week on writing simple IPO (input, processing and output) programs.

Here's the list of things you should do this week.

  1. H07 - Greeter: This is our first program with input. Much like the first console output programs you wrote, it displays a simple message. The difference it, this time when you ask the user for their name, you actually listen!

    Before you can complete this assignment, you'll need to understand the material in Section 3.1—Interactive Programs.


  2. More IPO Practice—TipCalc: This is a more complete console IPO program, named TipCalc that reads a subtotal and gratuity rate and then computes the gratuity and total bill.
  3. Writing Java Expressions: Often you'll be asked to write programs that compute different values, like converting Celsius to Fahrenheit, for instance. When you do that, the formula will be written using mathematical notation. Unfortunately, programming languages don't use the same notation, so you'll have to convert the formula into a Java expression, often using functions from the Math class.

    Follow the instructions in this assignment to convert different kinds of mathematical formulae into Java expressions. (This assignment was borrowed from the ICS 21 class at UCI.)
