CS 170—Chapter 8

Exercises and Assignments


In this chapter you learned how to write graphical appliations. Here's the list of things you should do this week.

  1. ACM Graphical Shapes—GraphicsHierarchy: Read Section 8.1—ACM Graphics Programs, and Section 8.2—Meet the GObject where you'll learn how to write programs using the ACM Graphics packages. Then, put what you've learned to work creating the ACM GraphicsProgram, named GraphicsHierarchy that draws a partial diagram of the acm.graphics class hierarchy.
  2. Images and Playing Cards—FiveCards: Read Section 8.3—Fillable and Scalable, and Section 8.4—GImage and Friends where you'll learn how to use the image-displaying classes in the ACM Graphics packages. Using the TwoCards example from Section 8.4 as a model, write write a program named FiveCards that deals five randomly selected playing cards face up on a table.
  3. Your First Widget—JLabelWorld: Read Section 8.5—Meet the Widgets where you'll learn how to use the interactors contained in Java's Swing widget package. Using the WidgetWorld example from Section 8.5 as a model, write write the program JLabelWorld to explore some of the features of the JLabel class. The handout has step-by-step instructions.
