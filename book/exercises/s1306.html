CS 170—Week 6

Exercises and Assignments


The big idea this week is writing functions that make decisions. You should know how to use the if and else statements, the logical operators and how to code different kinds of multi-way branching functions.

Here's the list of things you should do this week.

  1. H16—Writing a Logic Function: Read Section 6.3—Introducing Selection, and Section 6.4—Predicate Functions where you'll learn how to write functions that make decisions. Then, complete the function lastTwo() following the instructions in the handout.
  2. H17—Logical Operators: In Section 6.5—Logical Operators, you'll learn how to combine decisions, using Java's logical operators, AND and OR. In this assignment, you'll put your new-found knowlege to work, providing a software tool that the OCC squirel community can use to plan their weekend entertainment.
  3. H18—Multiple Choice: In Section 6.6—Multiple Choice, you'll learn about several different patterns that can be used to make decisions when there are more than two possible outputs. For instance; suppose you and your date want to have a night on the town, visiting a fancy restaurant. Will you get a table? Now, there's an app for that; you're going to write it.
  4. H19—Nested If Statements: You have a talking parrot. Unfortunately, you've had some difficulty training it, so that it knows when it's appropriate to vocalize. Your landlord, and fellow tenants are less than pleased. Write an app for that.
