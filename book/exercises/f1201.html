CS 170—Week 1

Homework and Exercises


The exercises this week are pretty simple, since we're just getting started, and because there's a lot of things you need to do to get set up. Here's the list of things you should do this week.

  1. Set Up Your Tools: If you're going to work at home, then follow the instructions in that handout I've provided to install Java along with the DrJava Integrated Development Environment. We'll go over most of these steps in class, so you only need to do this if you are going to work at home.
  2. H01 - Your First Console Program: This the exercise that goes along with the Java Mechanics lesson in your textbook.
  3. H02 - Your First GUI Program: Building a graphical program. Again, I've provided you with the code you need to type. The purpose of this exercise is to get more practice with the mechanical programming process.
  4. H03 - Your First Java Applet: Building a graphical program program that can be displayed in a brower. I've provided you with the code you need to type. The purpose of this exercise is to get more practice with the mechanical programming process.
  5. H04 - Using Windows: To make sure you're comfortable using Windows Explorer, I want you to complete this basic Windows Shell tutorial. Even if you're already a Windows guru, go through this to make sure you know how to complete all of the tasks listed.

    This assignment is not due until next week.
