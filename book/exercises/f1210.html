CS 170—Week 10

Exercises and Assignments


This week you started working with the collections of values, called arrays. You learned how to create arrays, how to pass arrays to methods, and how to write methods that return an array. You also learned how to use loops to process all of the elements in an array. The assignments this will will give you some practice doing each of these things.

Here's the list of things you should do this week.

  1. H30: Array Basics: Read Section 10.1—Primitive Arrays where you'll learn how to define array variables, instantiate arrays containing primitive elements, initialize arrays to predetermined values, and access individual array elements. Then, put what you've learned to work completing the three sections in BasicArrayProblems.java.
  2. H31: Arrays as Function Parameters: Read the first portion of Section 10.2—Arrays and Methods to learn how pass an array as input to your functions. Then, complete the sameFirstLast() problem.
  3. H32: Output Parameters—Swap Ends: Read the second portion of Section 10.2—Arrays and Methods to learn how use arrays as output parameters with your procedures. Then, complete the swapEnds() problem.
  4. H33: Returning Arrays: Read the final portion of Section 10.2—Arrays and Methods to learn how write functions that create and return arrays. Then, complete the makePI() problem.
  5. H34: Buttons, Text and Events—BillBoard: Read Section 10.5—Buttons and Events, and Section 10.6—Text Components, and then try out your skills creating this electronic BillBoard program. This assignment will be due at the first class meeting next week.
  6. H35: Widgets & Animation—Marquee: After you have Homework 34 working, turn your BillBoard into an animated Marquee. This assignment will be due at the first class meeting next week.
