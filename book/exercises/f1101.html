CS 170—Week 1

Exercises and Assignments


The exercises this week are pretty simple, since we're just getting started, and because there's a lot of things you need to do to get set up. Here's the list of things you should do this week.

  1. Introduce Yourself:

    Before we get started, I want you to introduce yourself to the other members of the class. To post your introduction, create a new message in the Blackboard Introductions discussion topic area, and use your last name, followed by your first name, as the subject line. You can see my posting for an example of how to do this. (Of course you're free to reply to other students introductions as well.)

    Then, go ahead and tell us a little about yourself. Here are some things you might want to cover:

    • What's your name? (although that should be in the subject)
    • How old are you?
    • Where are you from?
    • How long have you been at OCC?
    • What are your long-term educational goals?
    • Are you planning on transferring? What's your major?
    • What about your career goals?
    • Do you know anything about computer programming?
    • What other programming classes have you taken?
    • Why are you taking CS 170 and what do you hope to learn?
    • Any hobbies, pets, significant others, consuming passions?

    You don't have to answer every question, if you don't want. One way to introduce yourself is to think about what you'd like to know if you were meeting someone for the first time; you can sort of interview yourself!

    You can attach a picture if you like, but try to keep it small. Don't upload the 10MB picture from your digital camera without resizing it first.

  2. Using Windows: To make sure you're comfortable using Windows Explorer and the Windows command-line (which is what we'll use in the OCC Computing Center), Cay Horstmann has created a basic Windows Shell tutorial. You'll find it at:
    http://faculty.orangecoastcollege.edu/sgilbert/book/Win7ShellTutorial/index.html
    Even if you're already a Windows guru, I'd read through this to make sure you know how to complete all of the tasks listed. To get the most out of this assignment, just start a new file in your text editor or word processor and then answer each of the questions in the document. As always, you'll find solutions on the solutions page, so you can check your work.
  3. Set Up Your Tools: If you're going to work at home, then follow the instructions in that handout I've provided to install Java along with the DrJava Integrated Development Environment. We'll go over most of these steps in class, so you only need to do this if you are going to work at home.
  4. TicTacToeBoardPrinter: This is a "practice" exercise to get you ready for the exams. Try this after you've read the first chapter in the online text to see how much you've absorbed. To do this exercise, start DrJava (fresh, with no other apps open) and create a new console-mode application in the Chapter01 folder.
    • Name the program TicTacToeBoardPrinter.
    • The output of the program should look like this:
      +---+---+---+
|   |   |   |
+---+---+---+
|   |   |   |
+---+---+---+
|   |   |   |
+---+---+---+
      Don't put any spaces around your tic-tac-toe board. You should have exactly seven lines of output.

    Once you have created the program, compile it and run the Check program to see your score. If you didn't do as well as you thought, check the Solutions page online for some tips that may help you.

  5. StaircasePrinter: This is another "practice" exercise to get you ready for the exams. Create a new console-mode application in your Chapter01 folder.
    • Name the program StaircasePrinter.
    • The output of the program should look like this:
                  +---+
            |   |
        +---+---+
        |   |   |
    +---+---+---+
    |   |   |   |
+---+---+---+---+
|   |   |   |   |
+---+---+---+---+
      Don't put any extra spaces around your staircase. You should have exactly nine lines of output.

    Once you have created the program, compile it and run Check to see your score. Check the Solutions page online if you have difficulty.
