CS 170—Week 1

Homework and Exercises


The exercises this week are pretty simple, since we're just getting started, and because there's a lot of things you need to do to get set up. Here's the list of things you should do this week.

  1. H01 - Your First Console Program: This the exercise that goes along with the Java Mechanics lesson in your textbook.
  2. H02 - Your First GUI Program: Building a graphical program. Again, I've provided you with the code you need to type. The purpose of this exercise is to get more practice with the mechanical programming process.
  3. H05 - Printing a Message: This is the exercise that goes along with the Introduction to Strings lesson in your textbook. You'll be asked to print a program containing quotes and backslashes.
