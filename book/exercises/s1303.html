CS 170—Week 3

Homework and Exercises


This week you'll take Proficiency Quiz 2—Input, Processing and Output. To prepare, you'll need to write several interactive programs, concentrating on simple IPO: Input, Processing and Output.

Here's the list of things you should do this week.

  1. More IPO Practice—TipCalc: This is a more complete console IPO program, named TipCalc that reads a subtotal and gratuity rate and then computes the gratuity and total bill.
  2. Writing Java Expressions: Often you'll be asked to write programs that compute different values, like converting Celsius to Fahrenheit, for instance. When you do that, the formula will be written using mathematical notation. Unfortunately, programming languages don't use the same notation, so you'll have to convert the formula into a Java expression, often using functions from the Math class.

    Follow the instructions in this assignment to convert different kinds of mathematical formulae into Java expressions. (This assignment was borrowed from the ICS 21 class at UCI.)
