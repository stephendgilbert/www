CS 170 Introduction to Java Programming

Objects, Input and Processing


Welcome to Chapter 3. In the last two chapters, you learned how to perform output and create variables. In this chapter you'll learn how to write programs that create objects, get input, and that perform computations. After all, that's why they're called computers, right?.

In this chapter you'll learn:

  • how to write use the external ACM libraries to write more capable console programs.
  • how to create objects and send them messages
  • how to perform calculations to process the input
  • how to work with Math functions
  • how to create Color and Font objects

To the right you'll find the links to the individual sections in this chapter. On the exercises page you'll find links to those exercises and homework assignments that accompany this chapter.

Remember that during this compressed Summer session, the homework assignments are entirely for your benefit. You can check your work by using the Check button in DrJava, but you won't submit any of the assignments, and they will not contribute to your course grade.
