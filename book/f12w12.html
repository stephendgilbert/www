User-Defined Types


Welcome to Chapter 12. For the last few weeks you've worked with graphical objects, using classes in the AWT, Swing, and ACM libraries. Where, though, do such classes come from? The objects you've been using are not built into the Java language; instead, they were designed by programmers to meet a specific need.

In this chapter, you'll learn how to define your own classes, rather than just using objects that others have designed. Learning how to define your own types is really at the heart of Object-Oriented Programming.

In this chapter you'll:

  • create and define the fields to store object state.
  • write instance methods that provide object behavior.
  • use encapsulation to protect object data.
  • write constructors to initialize objects.
  • apply your new skills to the game of Pong
