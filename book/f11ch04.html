CS 170 Introduction to Java Programming

Characters, Strings and Functions


Welcome to Chapter 4. In this chapter, we're going to continue our IPO theme, performing more complex calculaions, learning how to use the char primitive type and it's more capable cousin, the String class.

Once you learn to use String functions, you'll be able to handle more complex input and produce more sophisticated output as well. Finally, we'll finish up this chapter with a little more sophisticated graphical output.

In this chapter you'll learn:

  • how to recognize different kinds of syntax errors.
  • how the different side-effect operators work.
  • to use the char primitive type
  • to work with String functions
  • to effectively use the Java documentation
  • to parse strings and convert them to numbers
  • to use the output-formatting functions.
  • to draw shapes and lines with Java's Graphics class
