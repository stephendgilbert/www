Welcome to Unit 5 This week you'll continue the application of computer logic as you learn about Java's logical operators, handle more complex decisions, and learn how to write more advanced loops.

In this unit you'll learn:

  • to combine Boolean conditions.
  • to implement multi-way decisions.
  • to use Java's while loop.
  • to write indefinite loops that test a condition.
  • to use jumps and iterators to process Strings and files.

A. Logical Operators
Learn how to combine Boolean expressions by using Java's logical operators.

B. Multiple Choice
Learn how to make decisions involving a mutltitude of different possible outcomes.

C. The while Loop
Learn how to use the while statement to write loops use a counter and those that test a condition.

D. Indefinite Loops
Learn how to use Java's do-while loop and how to manage necessary and intentional bounds.

E. Jumps and Iterators
Learn how to use iterators to process the characters in a String or the text in a file, and how to use Java's jump statements.

Exercises
Get some practice completing these exercises and assignments.
