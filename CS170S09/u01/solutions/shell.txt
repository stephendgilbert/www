Windows Shell Tutorial

Here are my solutions to each of the exercises in
Cay Horstmann's San Jose State Windows Shell tutorial.
Compare your answers, making any corrections, (or asking for
clarification on the discussion board).

Exercise 1

Which keystrokes are required to launch the Notepad program without
touching the mouse?

    Ctrl+Esc to open the Start menu, R to open the Run dialog,
    notepad to specify the program to run, and Enter to
    launch it.

Exercise 2

What is the advantage of using the Run dialog to launch the Notepad
program? Hint: Locate the Notepad program in the start menu tree. In which
submenu did you find it? How long did it take you to find it? Will it be
in the same position on your friend's computer?

    The advantage is that the computer "knows" where notepad
    is located, so I don't have to go looking for it on the
    menu. Also, even if I know where it is on the menu, it takes
    several mouse-clicks to locate it on the Accessories menu. And,
    as the hint points out, the program may be in a different location
    on a different computer.

Exercise 3

Run three programs, for example, a browser, the notepad, and the calc
program. Switch to the browser program if it is not currently active
(either with the mouse or with Alt+Tab).  Hit Alt+Tab and let go
immediately. Which program is selected now? Hit Alt+Tab again and let go
immediately. Which program is selected now? Why is the result surprising?
Why is it useful?

    When you press and release the Alt-Tab combination, it immediately
    goes back to the previously opened document. This is surprising
    because you'd expect the files to be accessed "linearly", that is,
    that Alt+Tab would take you to the next program, not the previous
    one. This behavior is useful, though, because it allows you to
    rapidly switch between two programs and use them "simultaneously".
    (You might, for instance, want to switch between your text editor
    and the command prompt for writing and compiling your Java
    programs.)


Exercise 4

Suppose you want to see as much information as possible in your browser
window. What keystrokes do you use to maximize the browser window?

    The Alt+SPACE command opens up the Window menu, where you can press
    X to maximize the window.

Exercise 5

Now switch to the browser window. If it doesn't currently show a document,
then visit your favorite web site. Use the mouse to select a phrase of
your choice. Your task is to copy the selected text to the Notepad
program, without further use of the mouse. What keystrokes to you use?

    Ctrl+C to copy to the clipboard. Alt-Tab and then Tab again to locate
    the Notepad program, Alt+V to past the text into notepad.

Exercise 6

exercise. If not, paste or type a couple of lines. Now highlight the
entire line containing the cursor, using as few keystrokes as possible.
What keystrokes to you use?

    The Home key to go to the beginning of the line, and then
    Shift+End to highlight to the end of the line.

Exercise 7

the next occurrence of Hello, with Match case checked? Keystrokes only, no
mouse!

    check the Match Case checkbox, and Alt+F to click the Find-Next button.

Exercise 39

Try the command dir c:\windows /p. What is the effect of the /p option?

    It displays the contents of the directory a
    screen at a time, waiting for you to press a key
    to advance to the next screen.

Exercise 40

What are all options of the del command?

  /P            Prompts for confirmation before deleting each file.
  /F            Force deleting of read-only files.
  /S            Delete specified files from all subdirectories.
  /Q            Quiet mode, do not ask if ok to delete on global wildcard
  /A            Selects files to delete based on attributes
  attributes    R  Read-only files            S  System files
                H  Hidden files               A  Files ready for archiving
                -  Prefix meaning not

Exercise 41

Copy this command: dir /oen /p c:\windows
into the shell window and execute it. What happens?

    I get a paused directory. The /o switch displays
    the files in sorted order. The e means that they are
    sorted by extension and the n means that they'll be
    sorted by name. (All of the .txt files will be together,
    and they'll be sorted by name)

Exercise 42

Execute the command: notepad c:\yourname\tutorial\myfile.txt
What happens? Explain.

    This command opens notepad and then tries to load
    the file myfile.txt. Since we don't have a myfile.txt
    (just sample.txt), notepad asks if we want to create it.
