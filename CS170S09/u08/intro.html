Welcome to Unit 8 This week, we'll continue working in the GUI world. Instead of just "drawing pictures", like you did back in Unit 3, this week our focus will be on writing graphical application programs, and making use of graphical objects.

We'll start by learning to use the objects that are defined in the ACM graphics package. Then you'll learn how to use several of Java's widgets and how respond to different kinds of events, so you can write interactive GUI programs. We'll finish up this week by learning how to animate your programs.

In this unit you'll learn:

  • how to create graphical applications, using the ACM GraphicsProgram class.
  • how to create and use the ACM graphical components.
  • how to use Java's event handling system to create programs that respond to mouse clicks.
  • how to use Java's widgets to make your GUI apps interactive.
  • how to animate the objects in your applications.
