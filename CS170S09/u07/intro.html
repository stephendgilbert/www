Welcome to CS 170, Unit 7. For the first half of this class you wrote text-based Java console applications, so you could concentrate on programming fundamentals.

During the second half of the class, we'll turn our attention to object-oriented programming or OOPs. We'll start out by learning to write graphical programs that run inside your Web browser, called applets.

This week you'll learn:

  • a little history of the Web
  • how to create and deploy Java applets
  • how to create and use Color objects
  • how to use accessor and mutator methods.
  • how to paint with Java's Graphics class

I've also provided an optional HTML lesson for those of you who want to know the basics of creating Web pages and integrating Java applets with your pages. You won't be tested on this lesson.

A. Java and the Web
Learn about computer networks, the Internet and the Web, and see where Java fits into the picture.

B. Java Applets
Learn about the syntax differences between applications and applets, and meet Java's Graphics class.

C. Objects and Colors
Create and use object variables using the Color class as your learning laboratory.

D. Fonts and Friends
Learn how to create Font objects and use accessor and mutator methods.

E. Applet Graphics
Learn how to use Graphics class to draw pictures using lines and shapes.

Exercises
Get some practice completing these exercises and assignments.

HTML Programming
Learn how to use the HyperText Markup Language to create Web pages and how to add Java applets to your pages.
