Welcome to Unit 11. This week you'll learn two different techniques for creating new classes: composition and inheritance. You'll also learn how to work with Java's ArrayList class and create and process two-dimensional arrays.

This week you'll learn:

  • how to create new classes from existing classes
  • how to use inheritance to extend the ACM and JFC components
  • how to combine classes to create new types
  • how to create new single-value user-defined types
  • how to use the ArrayList class to process object lists

A. Introducing Inheritance
Learn how to create new classes by extending existing classes.

B. Inheritance at Work
Learn how to use inheritance to create new versions of the ACM and JFC component classes.

C. Composite Objects
Learn how to use the technique of composition to combine existing classes to create new versions.

D. Enumerated Types
Create user-defined single-value types and use them to create your own PlayingCard class.

E. ArrayLists
Use encapsulation to hide the details of processing partially-filled arrays, and learn how to use Java's generic ArrayList class to store any type of element.

Exercises
Get some practice completing these exercises and assignments.
