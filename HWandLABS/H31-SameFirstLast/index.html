Array Parameters—Same First & Last


This week, you learned how to use methods with arrays. For this assignment, I want to to write a method that takes (and processes) and array as a parameter. Before you start this assignment, make sure you have read the first part of Section 10.2—Arrays and Methods, (up to Modifying Array Parameters).

Same First & Last

Write the function named sameFirstLast(). You'll find the file (SameFirstLast.java) in this week's code folder, not ArrayMethodProblems.java as shown in the help video.) Here are the specifications:

The function has one parameter: an array of int, and it returns true if the array has at least one element, and, the first and last elements are the same.

Here are the Javadoc tags:

  • @param nums the array of ints to examine.
  • @return true if the array is at least length one and the first and last elements are the same.

Here are some examples:

  • sameFirstLast({1, 2, 3}) returns false
  • sameFirstLast({1, 2, 3, 1}) returns true
  • sameFirstLast({1, 2, 1}) returns true
