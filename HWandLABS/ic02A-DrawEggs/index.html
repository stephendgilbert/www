Procedures


This week, you learned how to use the functions in the String class, (specifically, the length() and substring functions). You'll also learn how to write your own procedures and functions. Your fourth programming exam will require you to write your own simple String functions, while using the library functions you learned in Chapter 2.

Procdural Decomposition with DrawEggs

In lecture you learned how to break a single main() function into a series of procedures, eliminating redundancy. Let's see if you can do that on your own by writing a complete Java console program named DrawEggs that prints the pattern shown on the right. (Click the image to make it larger.)

Here are the requirements:

  • You should use at least at least three procedures, (not counting main()).
  • Do not do any printing inside your main() method. The only thing that main() should contain is procedure calls.
