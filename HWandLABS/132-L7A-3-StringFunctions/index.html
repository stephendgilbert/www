Drill & Practice

String Functions without Loops


In class tonight you reviewed the concept of string slicing and concatenation. Put that to working completing the following functions (most of which can be completed in a single line). Use the starter code in your Eclipse folder.

Make Tags

The web is built with HTML strings like "<i>Yay</i>" which draws Yay as italic text. In this example, the "i" tag makes and which surround the word "Yay". Write the function named makeTags(), which, given tag and word strings, returns the HTML string with tags around the word, e.g. "<i>Yay</i>".

Here are some examples:

  • makeTags("i", "Yay") returns "<i>Yay</i>"
  • makeTags("i", "Hello") returns "<i>Hello</i>"
  • makeTags("cite", "Yay") returns "<cite>Yay</cite>"

Extra End

Write the function named extraEnd(). Here are the specifications:

Given a string, return a new string made of 3 copies of the last 2 characters of the original string. The string length will be at least 2.

Here are some examples:

  • extraEnd("Hello") returns "lololo"
  • extraEnd("ab") returns "ababab"
  • extraEnd("Hi") returns "HiHiHi"

First Half

Use string slicing and the len() function to solve this. Given a string of even length, return the first half. So the string "WooHoo" yields "Woo".

Here are some examples:

  • firstHalf("WooHoo") returns "Woo"
  • firstHalf("HelloThere") returns "Hello"
  • firstHalf("abcdef") returns "abc"

Non Start

Write the function named nonStart(). Here are the specifications:

Given 2 strings, return their concatenation, except, omit the first character of each. The strings will be at least length 1.

Here are some examples:

  • nonStart("Hello", "There") returns "ellohere"
  • nonStart("java", "code") returns "avaode"
  • nonStart("shotl", "java") returns "hotlava"

Middle Two

Write the function named middleTwo(). Here are the specifications:

Given a string of even length, return a string made of the middle two chars, so the string "string" yields "ri". The string length will be at least 2.

Here are some examples:

  • middleTwo("string") returns "ri"
  • middleTwo("code") returns "od"
  • middleTwo("Practice") returns "ct"
