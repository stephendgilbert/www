Basic Logic Functions

Today we will concentrate on writing methods using only the if and else control structures. These are the most fundamental decision making skills that you'll need for Proficiency Quiz 5.

Even if you already know how to use nested if statements, for this assignment, try to solve these using only simple if statements along with the return statement. Don't use any of the binary Boolean operators like && or ||. You may use the Boolean not operator (!), even though we haven't covered it yet.

Makes Ten

Write the method named makes10(). Here are the specifications:

Given 2 ints, a and b, return true if one if one them is 10 or if their sum is 10.

Here are the Javadoc tags:

  • @param a is the first int value.
  • @param b is the second int value.
  • @return true if one is 10 or if their sum is 10.

Here are some examples:

  • makes10(9, 10) returns true
  • makes10(9, 9) returns false
  • makes10(1, 9) returns true
