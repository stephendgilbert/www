Escape Sequences


Now that you've had some practice with the writing programs using escape sequences, let's see if you can apply it to writing a simple program that displays String output. Your goal is to write this program from "scratch", which is what you'll need to do on the Programming Proficiency exams.

Spikey

Here is another "practice" exercise to try for output and escape sequences. Write a "regular" console-mode program named Spikey, (placed in your Chapter01 folder), that prints the output shown here:

As with the previous two homework assignments, the purpose is to make sure you know how to use escape sequences in your output. Notice that there are six lines of output with Spikey, and, there are no trailing spaces in the output.

Once you have created the program, compile it and run Check to see your score. If you didn't do as well as you thought you should, fix your errors and Check again. If you really get stuck, then this video may give you some ideas.
