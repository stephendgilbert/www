Homework Exercise

Salary Survey


In Section 2.3.2—Formatted Output, you're shown how to "fine-tune" your output using the facilites of the Formatter class and the System.out.printf() and String.format() methods. In this homework, you'll going to put these methods to work to display the output from an IPO program. You'll also need to remember how to parse String input, converting it to a double.

Salary Survey

Write a standard console program named SalarySurvey. Place it in the Week04 folder, since that's where the test programs are located. Here are the program's requirements:

  1. Ask the user to enter their first name and desired yearly salary, separated by a space.
  2. Use the Scanner's next() method to read the input into two String variables.
  3. Use the appropriate parsing method to convert the salary into a double and store it in a double variable.
  4. Use printf() or String.format() to display the result like this: 
    Name: Steve        Desired salary: $ 1,275,315.75
----+----|----+----|----+----|----+----|----+----|
  5. You can assume that the user will only enter valid input for the salary value. The spacing should be the same for amounts up to 9 million. Here's another example: 
    Name: BillyG       Desired salary: $         1.00
----+----|----+----|----+----|----+----|----+----|
