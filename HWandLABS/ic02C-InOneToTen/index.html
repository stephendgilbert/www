Combining Decisions

Get some extra practice with the logical operators by completing these predicate functions. Complete these problems using if statements along with the binary Boolean operators like && and ||, along with the not operator (!).

In One To Ten

Write the function named in1To10(). Here are the specifications:

Given a number n, return true if n is in the range 1..10, inclusive. Unless "outsideMode" is true, in which case return true if the number is less or equal to 1, or greater or equal to 10.

Here are the Javadoc tags:

  • @param n the number to compare.
  • @param outsideMode if true, change the way the calculation is done.
  • @return true if n meets the conditions above.

Here are some examples:

  • in1To10(5, false) returns true
  • in1To10(11, false) returns false
  • in1To10(11, true) returns true
