Digital Pictures

For this Lab exercise, you're going to work with your own digital picture, scaling it and drawing on it. Download your picture from Facebook, or take one with your phone.

Your Small Picture

In lecture you worked with the Picture class. For this lab you're going to create a small, framed version of your picture, with your name on it. Here are the instructions, and here's an example with one of my pictures.

  1. Copy your digital picture into DrJava's Week01 folder. Rename the picture so that it includes both your first and last name, and so that it's easy to enter into your source code.
  2. Open MySmallPicture.java in the Week01 folder.
  3. Create a Picture object using your digital picture. Don't use the FileChooser class; just use the name of the file (with no path) to create your picture. Leave the code I've supplied to calculate the full path to the picture.
  4. Scale the picture so that the finished size is 275 pixels wide and 200 pixels high. To see what that is in percentages you'll have to use the getWidth() and getHeight() methods on the original picture.

    Note: this requires you to do some calculations.

  5. Use createGraphics() to get a pen to paint with on the new, smaller picture. This pen will be type Graphics2D.
  6. Uncomment the line that smoothes the drawing.
  7. Draw a square frame around your picture. This may take repeated drawRect() or some creative fillRect statements.
  8. Draw your name at the bottom of your portrait, in the center.
  9. Save the picture in the same folder, using the same name, but with the addition of "-smaller". Modify the original fileName variable, and use the write() method.

Once you have created the program, compile it and run it. Only when it looks like you want, should you try to Check it.

Quick Reference

Here are the pieces of code you'll need to complete this assignment.

  1. Get a file name (provided you don't already have one):
    String fname = FileChooser.picAFile();
  2. Create a Picture object: Picture pic = new Picture(fname)
  3. Display a Picture object: pic.show(); or pic.explore();
  4. About a Picture: int width = pic.getWidth(); (or getHeight();)
  5. Scale a Picture: Picture smaller = pic.scale(.5, .5);
  6. Retrieve a pen: Graphics2D pen = pic.createGraphics();
  7. Change the pen's color: pen.setColor(Color.MAGENTA);
  8. Draw a rectangle: pen.drawRect(x, y, width, height);
  9. Draw text: pen.drawString("text", left, bottom);
  10. Save your picture: pic.write(path);
