Arrays & Methods


This week, you learned how to use methods with arrays. For these three assignments, I want to to write methods that take (and processes) arrays as parameter, or that return arrays. Before you start this assignment, make sure you have read Section 6.4—Using Arrays with Methods.

Same First & Last

Write the function named sameFirstLast(). You'll find the file (SameFirstLast.java) in this week's code folder, not ArrayMethodProblems.java as shown in the help video.) Here are the specifications:

The function has one parameter: an array of int, and it returns true if the array has at least one element, and, the first and last elements are the same.

Here are the Javadoc tags:

  • @param nums the array of ints to examine.
  • @return true if the array is at least length one and the first and last elements are the same.

Here are some examples:

  • sameFirstLast({1, 2, 3}) returns false
  • sameFirstLast({1, 2, 3, 1}) returns true

Swap Ends

Array parameters are different than primitive parameters, because array parameters are references. Even though the parameter is passed "by value", the value (or array reference), still refers to the original array that was created outside of the method.

That means that array parameters can be used as function outputs. By using output parameters, even a void procedure can return information to its caller.

Write the procedure named swapEnds(). You'll find the file (SwapEnds.java) in this week's code folder.) Here are the specifications:

The swapEnds() procedure is passed an array of ints named nums. You should swap the first and last elements in the array. Since this is a procedure, you won't have any return value. (The array length will be at least 1.)

Here are the Javadoc tags:

  • @param nums an array of int to be modified.
  • No @return because this is a procedure.

Here are some examples:

  • swapEnds({1, 2, 3, 4}) changes nums to {4, 2, 3, 1}
  • swapEnds({1, 2, 3}) changes nums to {3, 2, 1}
  • swapEnds({8, 6, 7, 9, 5}) changes nums to {5, 6, 7, 9, 8}

Returning Arrays

A procedure can modify one of its array parameters for output, but a function can actually create and return an entirely new array.

Write the function named makePI(). You'll find the file (MakePI.java) in this week's code folder.) Here are the specifications:

The function has no parameters, but returns the first three digits of PI as an int array.

Here are the Javadoc tags:

  • @return an int array containing the first three digits of PI.

Here is only one example:

  • makePi() returns the int[] array {3, 1, 4}
