Lab Exercises

Basic Logic Functions


Our first set of labs this week will concentrating on writing functions using only the if and else control structures. These are the most fundamental decision making skills that you'll need for Proficiency Quiz 5.

Here are three function exercises that will give you some practice.

Even if you already know how to use nested if statements, for this assignment, try to solve these using only simple if statements along with the return statement. Don't use any of the binary Boolean operators like && or ||. You may use the Boolean not operator (!), even though we haven't covered it yet.

Sum Double

Our first function requires you to calculate two different kinds of sums. Given two int values, return their sum. Unless the two values are the same; then return double their sum.

Here are the Javadoc tags:

  • @param a is the first int value.
  • @param b is the second int value.
  • @return their sum or double their sum if equal.

Here are some examples:

  • sumDouble(1, 2) returns 3
  • sumDouble(3, 2) returns 5
  • sumDouble(2, 2) returns 8

Diff 21

Write the function named diff21(). Here are the specifications:

Given an int n, return the absolute difference between n and 21, except return double the absolute difference if n is over 21.

Here are the Javadoc tags:

  • @param n is the int passed as a parameter.
  • @return difference between n and 21 or double if n > 21.

Here are some examples:

  • diff21(19) returns 2
  • diff21(10) returns 11
  • diff21(21) returns 0

Makes Ten

Write the function named makes10(). Here are the specifications:

Given 2 ints, a and b, return true if one if one them is 10 or if their sum is 10.

Here are the Javadoc tags:

  • @param a is the first int value.
  • @param b is the second int value.
  • @return true if one is 10 or if their sum is 10.

Here are some examples:

  • makes10(9, 10) returns true
  • makes10(9, 9) returns false
  • makes10(1, 9) returns true
