Arrays & Algorithms


Proficiency Quiz 8—Arrays and Loops, will require you to process one dimensional arrays, counting, summing, averaging and finding the extreme values. In the last homework, I walked you through using all of those to process the elements in an array. For these two assignments, I'm going to let you work it out by yourself.

Array Count 9

Write the function named arrayCount9(). Here are the specifications:

Given an array of ints, return the number of 9s in the array.

Here are the Javadoc tags:

  • @param nums the original array to process.
  • @return the number of 9s in the array.

Here are some examples:

  • arrayCount9({1, 2, 9}) returns 1
  • arrayCount9({1, 9, 9}) returns 2
  • arrayCount9({1, 9, 9, 3, 9}) returns 3

Test your function with the Check program.

Array 667

Write the function named array667(). Here are the specifications:

Given an array of ints, return the number of times that two 6's are next to each other in the array. Also count instances where the second "6" is actually a 7.

Here are the Javadoc tags:

  • @param nums the array to rotate.
  • @return the number of 6s followed by a 6 or 7.

Here are some examples:

  • array667({6, 6, 2}) returns 1
  • array667({6, 6, 2, 6}) returns 1
  • array667({6, 7, 2, 6}) returns 1

Test your function with the Check program.
