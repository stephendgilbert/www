Using and Writing Methods

Programming Exam 2 will require you to write your own simple String methods, while using the library methods you learned in Chapter 2. In HelloName and MakeOutWord, you saw how methods take input, process it, and produce new output. Put what you learned there to work, completing this method exercise.

Make Tags

This method is similar to MakeOutWord. If you've forgotten the steps to complete a method problem, then turn back to that exercise.

The web is built with HTML strings like "<i>Yay</i>" which draws Yay as italic text. In this example, the "i" tag makes and which surround the word "Yay". Write the method named makeTags(), which, given tag and word strings, returns the HTML string with tags around the word, e.g. "<i>Yay</i>".

Here are some examples:

  • makeTags("i", "Yay") returns "<i>Yay</i>"
  • makeTags("i", "Hello") returns "<i>Hello</i>"
  • makeTags("cite", "Yay") returns "<cite>Yay</cite>"
