Combining Decisions

Get some extra practice with the logical operators by completing these predicate functions. Complete these problems using if statements along with the binary Boolean operators like && and ||, along with the not operator (!).

Near Hundred

Write the function named nearHundred(). Here are the specifications:

Given an int n, return true if it is within 10 of 100 or 200. Note Math.abs(n) returns the absolute value of a number.

Here are the Javadoc tags:

  • param n is the int passed as a parameter.
  • @return true if n is within 10 of 100 or 10 of 200.

Here are some examples:

  • nearHundred(93) returns true
  • nearHundred(90) returns true
  • nearHundred(89) returns false
