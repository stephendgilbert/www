Using and Writing Methods

Extra End

Write the method named extraEnd(). Here are the specifications:

Given a string, return a new string made of 3 copies of the last 2 characters of the original string. The string length will be at least 2.

Here are some examples:

  • extraEnd("Hello") returns "lololo"
  • extraEnd("ab") returns "ababab"
  • extraEnd("Hi") returns "HiHiHi"
