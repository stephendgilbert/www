L59—Big Diff

Write the function named bigDiff(). Here are the specifications:

Given an array length 1 or more of ints, return the difference between the largest and smallest values in the array.

Here are the Javadoc tags:

  • @param nums the array to search.
  • @return the difference between the largest and smallest numbers in the array.

Here are some examples:

  • bigDiff({10, 3, 5, 6}) returns 7
  • bigDiff({7, 2, 10, 9}) returns 8
  • bigDiff({2, 10, 7, 2}) returns 8

L60—Is Everywhere

Write the function named isEverywhere(). Here are the specifications:

We'll say that a value is "everywhere" in an array if, for every pair of adjacent elements in the array, at least one of the pair is that value. Return true if the given value is everywhere in the array.

Here are the Javadoc tags:

  • @param nums the array to examine.
  • @param val value to check inside the array.
  • @return true if value appears in every adjacent pair.

Here are some examples:

  • isEverywhere({1, 2, 1, 3}, 1) returns true
  • isEverywhere({1, 2, 1, 3}, 2) returns false
  • isEverywhere({1, 2, 1, 3, 4}, 1) returns false

L61—Match Up

Write the function named matchUp(). Here are the specifications:

Given arrays nums1 and nums2, of the same length, for every element in nums1, consider the corresponding element in nums2 (at the same index). Return the count of the number of times that the two elements differ by 2 or less, but are not equal.

Here are the Javadoc tags:

  • @param nums1 the first array to examine.
  • @param nums2 the second array to examine.
  • @return number of times matching values meet conditions described.

Here are some examples:

  • matchUp({1, 2, 3}, {2, 3, 10}) returns 2
  • matchUp({1, 2, 3}, {2, 3, 5}) returns 3
  • matchUp({1, 2, 3}, {2, 3, 3}) returns 2
