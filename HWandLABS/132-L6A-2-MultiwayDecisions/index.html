Multi-way Branches


You can combine decisions by using the logical operators, as you saw in the last set of lab exercises. You can do something similar by implementing multi-way branches, using sequential if statements, or by nesting.

Complete these problems using sequential or nested if statements. Do not use the logical operators, like and or or.

Red Ticket

Write the function named redTicket(). Here are the specifications:

You have a red lottery ticket showing ints a, b, and c, each of which is 0, 1, or 2. If all of the numbers have the value 2, then the payout is 10. Otherwise, if they are all the same, the payout is 5. Otherwise, so long as both b and c are different from a, then the payout is 1. Otherwise, the payout is 0.

Here are some examples:

  • redTicket(2, 2, 2) returns 10
  • redTicket(2, 2, 1) returns 0
  • redTicket(0, 0, 0) returns 5

Lucky Sum

Write the function named luckySum(). Here are the specifications:

Given 3 int values, a b c, return their sum. However, if one of the values is 13 then it does not count towards the sum and values to its right do not count. So for example, if b is 13, then both b and c do not count.

Here are some examples:

  • luckySum(1, 2, 3) returns 6
  • luckySum(1, 2, 13) returns 3
  • luckySum(1, 13, 3) returns 1

Evenly Spaced

Write the function named evenlySpaced(). Here are the specifications:

Given three ints, a b c, one of them is small, one is medium and one is large. Return true if the three values are evenly spaced, so the difference between small and medium is the same as the difference between medium and large.

Here are some examples:

  • evenlySpaced(2, 4, 6) returns True
  • evenlySpaced(4, 6, 2) returns True
  • evenlySpaced(4, 6, 3) returns False
