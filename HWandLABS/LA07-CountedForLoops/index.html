Counted for Loops


This week you learned how to repeat a set of actions, by using loops. There are several different kinds of loops, but the simplest are those that use a counter to control the number of repetitions. We call these counter-controlled loops, (or counted loops). The easiest way to write a counter-controlled loop is by using Java's for statement.

In these three labs, you'll put the for loop to work solving some String functions. To get started, make sure you have read Section 7.2—The for Loop, and completed the homework Double Char. That will prepare you to work on these problems.

String Times

Write the function named stringTimes(). Here are the specifications:

Given a String str and a non-negative int n, return a new String that is n copies of the original String.

Here are the Javadoc tags:

  • @param str the String to duplicate.
  • @param n the the number of times to duplicate str.
  • @return a String that is n copies of the original.

Here are some examples:

  • stringTimes("Hi", 2) returns "HiHi"
  • stringTimes("Hi", 3) returns "HiHiHi"
  • stringTimes("Hi", 1) returns "Hi"

String Explosion

Write the function named stringSplosion(). Here are the specifications:

The function takes one parameter, a non-empty String str (such as "Code") and returns a String in the form "CCoCodCode". Notice that this includes the first character of the original String, followed by the first two characters, and so on until the whole String is used.

Here are the Javadoc tags:

  • @param str the input String to process.
  • @return a new String as described above.

Here are some examples:

  • stringSplosion("Code") returns "CCoCodCode"
  • stringSplosion("abc") returns "aababc"
  • stringSplosion("x") returns "x"

Test your function with the Check program. There is no video for this problem.

String E

Write the function named stringE(). Here are the specifications:

Return true if the given String contains betwee 1 and 3 'e' characters (inclusive). Return false otherwise.

Here are the Javadoc tags:

  • @param str the String to process.
  • @return true if str contains between 1 and 3 'e' characters.

Here are some examples:

  • stringE("Hello") returns true
  • stringE("Heelle") returns true
  • stringE("Heelele") returns false
