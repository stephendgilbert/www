Lab 52, 53 & 54

Arrays & Methods


Profiency Quiz 7—Basic Arrays, will require you to create, initialize and access the elements of one-dimensional arrays. You will also be asked to write methods that accept arrays as parameters, and that return information by creating a new array, or by using an array parameter for output.

For this lab, you'll solve three problems that require arrays and methods. Note, none of these problems require you to write loops; you'll do that for PQ08.

L52—Mid Three

Write the function named midThree(). Here are the specifications:

Given an array of ints, (of odd length), return a new array length 3 containing the elements from the middle of the array. The array length will be at least 3.

Here are the Javadoc tags:

  • @param nums the original array to process.
  • @return a new array as specified above.

Here are some examples:

  • midThree({1, 2, 3, 4, 5}) returns {2, 3, 4}
  • midThree({8, 6, 7, 5, 3, 0, 9}) returns {7, 5, 3}
  • midThree({1, 2, 3}) returns {1, 2, 3}

L53—Rotate Left 3

Write the function named rotateLeft3(). Here are the specifications:

Given an array of ints of length 3, return an array with the elements "rotated left" so passing {1, 2, 3} returns an new array containing {2, 3, 1}

Here are the Javadoc tags:

  • @param nums the array to rotate.
  • @return a new, rotated array. Do not modify the original array.

Here are some examples:

  • rotateLeft3({1, 2, 3}) returns {2, 3, 1}
  • rotateLeft3({5, 11, 9}) returns {11, 9, 5}
  • rotateLeft3({7, 0, 0}) returns {0, 0, 7}

Test your function with the Check program. I have a video solution for a similar problem (on CodingBat), but this solution will not pass the checks. For this problem, you must make a new array, and not modify the array parameter (as the video does).

L54—Make Last

Write the function named makeLast(). Here are the specifications:

Given an int array, return a new array with double the length of the original where its last element is the same as the original array, and all the other elements are 0. The original array will be length 1 or more.

Here are the Javadoc tags:

  • @param nums the original array to process.
  • @return a new array as specified above.

Here are some examples:

  • makeLast({4, 5, 6}) returns {0, 0, 0, 0, 0, 6}
  • makeLast({1, 2}) returns {0, 0, 0, 2}
  • makeLast({3}) returns {0, 3}
