Drawing Shapes

Let's get some hands-on practice with writing procedures, breaking a big problem into little ones.

Here is something you can try:

  1. Run the console-mode program named DrawShapes, which you can find in your Week03 folder. As you can see, it draws a diamond, followed by an X, followed by a rocket on the screen, that looks like the snapshot at the right.
