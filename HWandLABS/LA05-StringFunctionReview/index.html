More String Functions


Let's finish up our labs this week by concentrating on skills you'll need for Proficiency Quiz 4. Here's what you must be able to do:

  • Given a specification showing the inputs and outputs of a functions, you must be able to write a syntactically correct "skeleton" that you can compile and test.
  • You must know how to use the String functions length(), substring(), trim(), and possibly charAt() to process an input string and construct an output string.

Here are three last function exercises that will give you some practice.

First Half

Our first function requires you to use the substring() and length() functions. Given a string of even length, return the first half. So the string "WooHoo" yields "Woo".

Here are the Javadoc tags:

  • @param str the input string (which will be of even length).
  • @return the first half of the input string.

Here are some examples:

  • firstHalf("WooHoo") returns "Woo"
  • firstHalf("HelloThere") returns "Hello"
  • firstHalf("abcdef") returns "abc"

Non Start

Write the function named nonStart(). Here are the specifications:

Given 2 strings, return their concatenation, except, omit the first character of each. The strings will be at least length 1.

Here are the Javadoc tags:

  • @param a the first input string
  • @param b the second input string
  • @return the concatenation of a and b, less the first character of each.

Here are some examples:

  • nonStart("Hello", "There") returns "ellohere"
  • nonStart("java", "code") returns "avaode"
  • nonStart("shotl", "java") returns "hotlava"

Middle Two

Write the function named middleTwo(). Here are the specifications:

Given a string of even length, return a string made of the middle two chars, so the string "string" yields "ri". The string length will be at least 2.

Here are the Javadoc tags:

  • @param str the input string
  • @return the middle two characters.

Here are some examples:

  • middleTwo("string") returns "ri"
  • middleTwo("code") returns "od"
  • middleTwo("Practice") returns "ct"
