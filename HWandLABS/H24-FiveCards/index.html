Five Cards


In Section 8.4, you learned how to use the GImage class to display playing cards on a simulated table. Using that section as a model, expand your program to display five, face-up, randomly selected playing cards, in the program FiveCards.

Images and Playing Cards—FiveCards

Using the TwoCards example from Section 8.4 as a model, write write a program named FiveCards that deals five cards face up on a table.

  • Display them nicely either side-by-side or slightly overlapped.
  • The cards are available inside the cards folder in this chapter's code folder.
  • Use the RandomGenerator() class to select random cards to deal.

Picking a Random Card: Create a single String variable containing the names of all of the card images like: "2c3c4c...". Then, generate a random number and then use the random number to pick the element to display, by using the substring() method. If cards is the name of your String variable, and cardNo is a random integer between 0 and 52, you can pick a random card with cards.substring(cardNo * 2, cardNo * 2 + 2).

Note that the program is not interactive. All cards can be dealt face up. They should not flip over like you did with TwoCards. Do NOT place any waitForClick() statements in your program, or the Check program will not be able to run.
