GHappy

You've previously completed several exercises using the for and while loops to repeat a set of actions. Now, you've added sentinel, flag-controlled and different jump statements to the mix.

You can complete this problem using any loops you like. You may need multiple loops. You may also make use of the the different jump statements, ike break and continue.

Solve this problem with loops, not with regular expressions and the replaceAll() method.

Don't Worry; G Happy!

Write the method named gHappy(). Here are the specifications:

We'll say that a lowercase 'g' in a string is "happy" if there is another 'g' immediately to its left or right. Return true if all the g's in the given string are happy.

Here are the Javadoc tags:

  • @param str the String to search.
  • @return true if all lowercase 'g's are happy.

Here are some examples:

  • gHappy("xxggxx") returns true
  • gHappy("xxgxx") returns false
  • gHappy("xxggyygxx") returns false
