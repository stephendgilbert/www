Integers & Strings

In class you learned how to read the Java documentation. You also learned how to convert Strings into integers, by parsing. In this homework, you'll going to explore a little more of the Integer class. To complete the assignment, look up the Integer class documentation, using either DrJava (as shown in class), or the link on the top of the syllabus page.

Integers and Strings

Write a standard console program named IntegersAndStrings. Place it in the Week01 folder, since that's where the test programs are located. Here are the program's requirements:

  1. Ask the user to enter an integer and then read and store the results in a variable. Use the Scanner nextInt() method.
  2. Use the methods in the Integer class to print the value of the variable in binary, octal and hexadecimal. You'll need to look them up the appropriate methods in the Javadocs.
  3. Print the maximum and minimum integer values by using the constants defined in the Integer class.
  4. Now, ask the user to enter two integers. Read and store the input in a pair of String variables, using the next() method. Then, convert the Strings to ints, add them together and print the results. Use concatenation, but do not format the output numbers.
