Inside your program's main() method, here are the basic steps you should follow:

  1. Create a Scanner object for input. Remember you'll have to import the package that contains the class.
  2. Prompt the user for input. You can see from the screenshot what the prompt should say. Remember that you don't want a newline at the end of the prompt.
  3. Read the user's input and store it in a double variable.
  4. Convert the input to fahrenheit using the formula above. Store the answer in a second double variable.
  5. Combine the calculated output with the appropriate verbiage and display it. You can see the output on the screenshot.

Remember that algebraic formulas often are written using integers or fractions, but actually expect real-number results. For instance, in the formula for this problem, you need to multiply the Celsius value by 9/5, but if you transcribe the formula literally, you'll get integer division, which always returns the value 1. Instead you need to use double values in your calculation.
