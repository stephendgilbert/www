Combining Decisions

Get some extra practice with the logical operators by completing these predicate functions. Complete these problems using if statements along with the binary Boolean operators like && and ||, along with the not operator (!).

Red Ticket

Write the function named redTicket(). Here are the specifications:

You have a red lottery ticket showing ints a, b, and c, each of which is 0, 1, or 2. If all of the numbers have the value 2, then the payout is 10. Otherwise, if they are all the same, the payout is 5. Otherwise, so long as both b and c are different from a, then the payout is 1. Otherwise, the payout is 0.

Here are the Javadoc tags:

  • @param a first number on the ticket.
  • @param b second number on the ticket.
  • @param c third number on the ticket.
  • @return payout (integer) as described above.

Here are some examples:

  • redTicket(2, 2, 2) returns 10
  • redTicket(2, 2, 1) returns 0
  • redTicket(0, 0, 0) returns 5
