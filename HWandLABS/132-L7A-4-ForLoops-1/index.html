Counting and Finding with the for Loop


These exercises should look familiar, since we've alreayd solved them with the while loop. Now, le's use the for loop, along with string slicing, to solve the same problems.

Cats and Dogs

Write the function named catDog2(). Here are the specifications:

Given a String str, return true if the words "cat" and "dog" appear the same number of times in the string.

Here are some examples:

  • catDog2("catdog") returns true
  • catDog2("catcat") returns false
  • catDog2("1cat1cadodog") returns true

Counting Code

Here's something a little harder. Write the function named countCode2(). Here are the specifications:

Given a String str, return the number of times that the word "code" appears anywhere in str, except we'll accept any letter for the 'd', so "cope" and "cooe" both count.

Here are some examples:

  • countCode2("aaacodebbb") returns 1
  • countCode2("codexxcode") returns 2
  • countCode2("cozexxcope") returns 2

Is Bob There?

Now, let's look at searching instead of counting. Write the function named bobThere2(). Here are the specifications:

Given a String str, return true if the given string contains a "bob" string. As with the countCode() function, we really don't care aobut the middle 'o' character, which can be anything character at all.

Here are some examples:

  • bobThere2("abcbob") returns true
  • bobThere2("b9b") returns true
  • bobThere2("bac") returns false
