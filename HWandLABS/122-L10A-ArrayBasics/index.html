CS 122 Lab Exercise

Array Basics


Last week, you started working with Java's array types. An array is a collection of variables that share a common name. In this lab assignment, I want you to practice doing three things:

  • Creating array variables, and instantiating the actual array that your variable refers to.
  • Initializing array elements with a set of predefined values, using the initialization syntax.
  • Use subscripting to store and retrieve selected elements from an array.

You'll complete this assignment in three steps. For each step, you'll need to read a portion of Section 6.1—Arrays in your textbook.

Array Basics

Open BasicArrayProblems (which you'll find in this chapter's code folder). Complete the first section of the program by defining the array instance variables fa, ia, da and ca, and instantiating the requested array objects.

Each definition should appear on one line and should include both the array definition and the instantiation as shown in the text. Once you are done, run the Check program to make sure you've defined the arrays correctly. (Only the first portion will check correctly.) Watch this short video if you get stuck.

Part II—Explicit Initialization

Locate the second section in BasicArrayProblems and define the initialized arrays monthNames, monthDays and oddNums.

Don't use the new operator; use the initialization list syntax instead. Once you are done, run the Check program to make sure you've defined the arrays correctly. Watch the video here if you get stuck.

Part III—Array Element Access

Let's try one last exercise in BasicArrayProblems.java. At the bottom of the file, write the lastMonth() function. I've already created a function skeleton that just returns an empty String.

Now that you know how to access array elements, modify it to return the last month of the monthNames array. Careful, though, that you don't hard-code the index value.
