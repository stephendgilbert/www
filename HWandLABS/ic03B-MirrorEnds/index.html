Mirror Ends

You've previously completed several exercises using the for and while loops to repeat a set of actions. Now, you've added sentinel, flag-controlled and different jump statements to the mix.

You can complete this problem using any loops you like. You may need multiple loops. You may also make use of the the different jump statements, ike break and continue.

Solve this problem with loops, not with regular expressions and the replaceAll() method.

Mirror Ends

Write the method named mirrorEnds(). Here are the specifications:

Given a String str, look for a mirror image (backwards) string at both the beginning and end of the given string. In other words, zero or more characters at the very beginning of the given string, and at the very end of the string in reverse order (possibly overlapping). For example, the string "abXYZba" has the mirror end "ab".

Here are the Javadoc tags:

  • @param str the String to search.
  • @return the string that is mirrored at the end.

Here are some examples:

  • mirrorEnds("abXYZba") returns "ab"
  • mirrorEnds("abca") returns "a"
  • mirrorEnds("aba") returns "aba"
