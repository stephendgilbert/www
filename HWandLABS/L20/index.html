L21: Using String Functions

This week, you learned how to use two more String functions. To try out your new expertise:

  1. Write (from "scratch") a regular (non-AMC) IPO console program named ProcessName.
  2. Ask the user to enter their name in the form (First MI. Last).
  3. Read the name into a single String variable, using nextLine(), not next().
  4. Use String functions to break the name into the first, last and middle initial.
  5. Print the resulting name rearranged and labeled in the form "Last, First MI." (including the quotes as shown below).

You can use the String functions substring(), indexOf(), lastIndexOf(), charAt() and trim() to solve this problem.

You may assume:

  • that every name will have exactly one word for the first and last names and a single character, followed by a period for the middle initial.
  • There will be at least one space separating each of the portions of the input name, but there may be more spaces.

Your program should handle spaces before the first name and after the last name as well. The program should look similar to this when it runs. (Just ignore the ACM console window in the picture. You will run it in DrJava as a standard console program.)
