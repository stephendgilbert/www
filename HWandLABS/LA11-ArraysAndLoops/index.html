Arrays & Loops


Here are some more practice exercises for Profiency Quiz 8—Arrays and Loops. Each of them require you to loop through an array and examine the elements before or after the current one, or keep track of how many times you've seen a particular value.

Array Front 9

Write the function named arrayFront9(). Here are the specifications:

Given an array of ints, return true if one of the first 4 elements in the array is a 9. Warning: the array length may be less than 4.

Here are the Javadoc tags:

  • @param nums the array to search.
  • @return true if a 9 is found in the first four elements.

Here are some examples:

  • arrayFront9({1, 2, 9, 3, 4}) returns true
  • arrayFront9({1, 2, 3, 4, 9}) returns false
  • arrayFront9({1, 2, 3, 4, 5}) returns false

No Triples

Write the function named noTriples(). Here are the specifications:

Given an array of ints, we'll say that a triple is a value appearing 3 times in a row in the array. Return true if the array DOES NOT contain any triples.

Here are the Javadoc tags:

  • @param nums the array to examine.
  • @return true if the array is "triple free" as described.

Here are some examples:

  • noTriples({1, 1, 2, 2, 1}) returns true
  • noTriples({1, 1, 2, 2, 2, 1}) returns false
  • noTriples({1, 1, 1, 2, 2, 2, 1}) returns false

Has 271

Write the function named has271(). Here are the specifications:

Given an array of ints, return true if it contains a 2, 7, 1 pattern—a value, followed by the value plus 5, followed by the value minus 1. Additionally the 271 counts even if the "1" differs by 2 or less from the correct value.

Here are the Javadoc tags:

  • @param nums the array to examine.
  • @return true if the array contains the "271" pattern.

Here are some examples:

  • has271({1, 2, 7, 1}) returns true
  • has271({1, 2, 8, 1}) returns false
  • has271({2, 7, 1}) returns true
