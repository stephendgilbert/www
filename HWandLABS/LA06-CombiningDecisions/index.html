Combining Decisions


You can get some extra practice with the logical operators by completing these predicate functions. To get started, make sure you have read Section 6.4—Logical Operators, and completed Cigar Party. That will get you started so you can continue with these problems.

Here are three function exercises that will give you some practice.

Complete these problems using if statements along with the binary Boolean operators like && and ||, along with the not operator (!).

Near Hundred

Write the function named nearHundred(). Here are the specifications:

Given an int n, return true if it is within 10 of 100 or 200. Note Math.abs(n) returns the absolute value of a number.

Here are the Javadoc tags:

  • param n is the int passed as a parameter.
  • @return true if n is within 10 of 100 or 10 of 200.

Here are some examples:

  • nearHundred(93) returns true
  • nearHundred(90) returns true
  • nearHundred(89) returns false

Red Ticket

Write the function named redTicket(). Here are the specifications:

You have a red lottery ticket showing ints a, b, and c, each of which is 0, 1, or 2. If all of the numbers have the value 2, then the payout is 10. Otherwise, if they are all the same, the payout is 5. Otherwise, so long as both b and c are different from a, then the payout is 1. Otherwise, the payout is 0.

Here are the Javadoc tags:

  • @param a first number on the ticket.
  • @param b second number on the ticket.
  • @param c third number on the ticket.
  • @return payout (integer) as described above.

Here are some examples:

  • redTicket(2, 2, 2) returns 10
  • redTicket(2, 2, 1) returns 0
  • redTicket(0, 0, 0) returns 5

In One To Ten

Write the function named in1To10(). Here are the specifications:

Given a number n, return true if n is in the range 1..10, inclusive. Unless "outsideMode" is true, in which case return true if the number is less or equal to 1, or greater or equal to 10.

Here are the Javadoc tags:

  • @param n the number to compare.
  • @param outsideMode if true, change the way the calculation is done.
  • @return true if n meets the conditions above.

Here are some examples:

  • in1To10(5, false) returns true
  • in1To10(11, false) returns false
  • in1To10(11, true) returns true
