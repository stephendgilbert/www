Counted for Loops

This week you learned how to repeat a set of actions, by using loops. There are several different kinds of loops, but the simplest are those that use a counter to control the number of repetitions. We call these counter-controlled loops, (or counted loops).

The easiest way to write a counter-controlled loop is by using Java's for statement. Put the for loop to work solving this String problem.

String Explosion

Write the method named stringSplosion(). Here are the specifications:

The method takes one parameter, a non-empty String str (such as "Code") and returns a String in the form "CCoCodCode". Notice that this includes the first character of the original String, followed by the first two characters, and so on until the whole String is used.

Here are the Javadoc tags:

  • @param str the input String to process.
  • @return a new String as described above.

Here are some examples:

  • stringSplosion("Code") returns "CCoCodCode"
  • stringSplosion("abc") returns "aababc"
  • stringSplosion("x") returns "x"
