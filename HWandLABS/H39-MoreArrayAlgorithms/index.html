More Array Algorithms


Proficiency Quiz 8—Arrays and Loops, will require you to process one dimensional arrays, counting, summing, averaging and finding the extreme values. Here are two more assignments to give you a little more practice.

Sum 13

Write the function named sum13(). Here are the specifications:

Return the sum of the numbers in the array, returning 0 for an empty array. Except the number 13 is very unlucky, so it does not count, and numbers that come immediately after a 13 also do not count.

Here are the Javadoc tags:

  • @param nums the array to process.
  • @return the sum except for 13s and numbers following.

Here are some examples:

  • sum13({1, 2, 2, 1}) returns 6
  • sum13({1, 1}) returns 2
  • sum13({1, 2, 2, 1, 13}) returns 6

Has 22

Write the function named has22(). Here are the specifications:

Given an array of ints, return true if the array contains a 2 next to a 2, anywhere in the array.

Here are the Javadoc tags:

  • @param nums the array examine.
  • @return true if there are two 2s together in the array.

Here are some examples:

  • has22({1, 2, 2}) returns true
  • has22({1, 2, 1, 2}) returns false
  • has22({2, 1, 2}) returns false
