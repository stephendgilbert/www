Counting and Finding

Now that you know how to repeat a set of actions, let's put the for loop to work by finding and counting things. To do this, you'll have to marry the for loop with one or more if statements.

Cats and Dogs

Write the function named catDog(). Here are the specifications:

Given a String str, return true if the words "cat" and "dog" appear the same number of times in the string.

Here are the Javadoc tags:

  • @param str the String to examine.
  • @return true if there ar the same number of cats and dogs.

Here are some examples:

  • catDog("catdog") returns true
  • catDog("catcat") returns false
  • catDog("1cat1cadodog") returns true

Test your function with the Check program. If you run into difficulty, you may want to watch one of these two videos.
