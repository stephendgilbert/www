Bill Board


In Section 10.5 and Section 10.6, you learned how to create buttons and textfields and how to respond to button clicks to carry out an action. Read both of those sections (working out each of the in-chapter exercises as you do), and then put your skills to work building a simple electronic "Billboard" application.

Buttons, Text-fields and Events—BillBoard

Create an ACM Graphics program named BillBoard that displays a message. Your program will have widgets to allow the user to enter a new message and to change the color of the display. Here's what it looks like as it runs:

Here are the instructions:

  • Making the program 600 wide and 200 high. Center the message vertically on the screen. To set the size for an ACM graphical application, just define these two instance variables as part of your class:
    public static final int APPLICATION_WIDTH = 600;
    public static final int APPLICATION_HEIGHT = 200;
  • Use a GLabel (not a JLabel) for the message. You'll also need two JButtons and a JTextField as shown in the example here.
  • When you position your label, center it horizontally.
  • When the Show It! button is pressed, grab the text from the JTextField and use it to change the message displayed in your label. When you do this, you need to re-center the GLabel Note: you should not create a new GLabel object when you do this. Just call the object's methods to change its location and contents.
  • When the Color button is pressed, select a new color. You can use an array of colors and cycle through them, or you can choose a color at random. Remember that the ACM RandomGenerator class has a nextColor() method that will do this for you.
