IPO Programs & Remainder

Your first programming exam involves IPO (input-processing-output) style programs. Here is a lab problem that requires you to use the remainder operator to break a number into pieces and then add the pieces.

SumDigits

Complete the starter program named SumDigits.java, which you'll find in the Week01 folder. The program reads a single integer from the user. (Do not read the input as a String.) You can assume that the user will cooperate and only input values between 0 and 1,000.

Your program should use the arithmetic operators discussed in the textbook (and in class) to sum all of the individual digits in the number. Here is a sample interaction so you can see what the program should look like as it runs.

To calculate the digits you'll have to do several divisions and integer remainder (%) operations. Notice that for any integer, n % 10 returns the value of the right-most digit, and n / 10 returns a new number with the right-most digit discarded.
