Returning Arrays


A procedure can modify one of its array parameters for output, but a function can actually create and return an entirely new array. To see how this works, read the final part of Section 10.2—Arrays and Methods, (the portion that begins with Returning Array Variables). Then, complete this simple exercise.

Returning Arrays

Write the function named makePI(). You'll find the file (MakePI.java) in this week's code folder.) Here are the specifications:

The function has no parameters, but returns the first three digits of PI as an int array.

Here are the Javadoc tags:

  • @return an int array containing the first three digits of PI.

Here is only one example:

  • makePi() returns the int[] array {3, 1, 4}
