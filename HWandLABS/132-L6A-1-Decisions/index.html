Combining Decisions with Logical Operators


Complete these problems with the binary Boolean operators like and and or, along with the not operator. You should not need any if statements.

Near Hundred

Write the function named nearHundred(). Here are the specifications:

Given an integer n, return True if it is within 10 of 100 or 200. Note abs(n) returns the absolute value of a number.

Here are some examples:

  • nearHundred(93) returns True
  • nearHundred(90) returns True
  • nearHundred(89) returns False

In One To Ten

Write the function named in1To10(). Here are the specifications:

Given a number n, return true if n is in the range 1..10, inclusive. Unless "outsideMode" is true, in which case return true if the number is less or equal to 1, or greater or equal to 10.

Here are some examples:

  • in1To10(5, False) returns True
  • in1To10(11, False) returns False
  • in1To10(11, True) returns True
