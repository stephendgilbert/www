Lab Exercises

Counting and Finding


Now that you know how to repeat a set of actions, let's put the for loop to work by finding and counting things. To do this, you'll have to marry the for loop with one or more if statements.

To get started, make sure you have read Section 7.2—The for Loop, and completed the Counting Greetings homework. That will prepare you to work on these problems.

Cats and Dogs

Write the function named catDog(). Here are the specifications:

Given a String str, return true if the words "cat" and "dog" appear the same number of times in the string.

Here are the Javadoc tags:

  • @param str the String to examine.
  • @return true if there ar the same number of cats and dogs.

Here are some examples:

  • catDog("catdog") returns true
  • catDog("catcat") returns false
  • catDog("1cat1cadodog") returns true

Test your function with the Check program. If you run into difficulty, you may want to watch one of these two videos. (While both of these videos are very similar, there are small differences in the way the problem is solved. One uses returns a boolean expression, while the other uses an if statement, for instance):

Remember that many of my videos for the function exercises that we will do in this class make use of the Codingbat.com Web site put up by Nick Parlante at Stanford. When you do the exercise in DrJava, you will need to write the method heading yourself, something that Coding.bat does for you.

Counting Code

Here's something a little harder. Write the function named countCode(). Here are the specifications:

Given a String str, return the number of times that the word "code" appears anywhere in str, except we'll accept any letter for the 'd', so "cope" and "cooe" both count.

Here are the Javadoc tags:

  • @param str the String to process.
  • @return the number of times "co?e" appears in str.

Here are some examples:

  • countCode("aaacodebbb") returns 1
  • countCode("codexxcode") returns 2
  • countCode("cozexxcope") returns 2

Test your function with the Check program. If you run into difficulty, you may want to watch one of these two videos.
