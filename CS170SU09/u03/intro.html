Welcome to CS 170 Unit 3. In this unit, we're going to continue working with Java's fundamental data types and the functions in the Math and String classes. You'll also learn how to write programs that contain multiple procedures and use that skill to make your programs simpler.

In this unit you'll learn:

  • how the different side-effect operators work
  • how to use functions in the Math class
  • how to work with String functions
  • how to do output formatting
  • how to convert between strings and numbers
  • how to write programs with multiple procedures

A. Side Effects
Learn about the Java side-effect operators that modify their operands as they work.

B. Math and Characters
Learn to use the functions in Java's Math classes and meet the char primitive type.

C. More on Strings
Learn to use the Java API documenation and explore the methods avaliable to String objects.

D. Formatting and Parsing
Learn how to format numeric and String output, and how to convert between String and numeric types.

E. Writing Procedures
Learn how to apply functional decomposition to solve complext problems by writing your own methods.

Exercises
Get some practice completing these exercises and assignments.
