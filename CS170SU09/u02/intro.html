Welcome to Unit 2 In the last unit you learned about the basic structure of a Java program. In this unit you'll learn how to create and manipulate variables, using Java's primitive numeric types, as well as learning how to write programs that perform computations. After all, that's why they're called computers, right?.

In this unit you'll learn:

  • how to use Java's system output objects
  • how to create and use numeric variables
  • how to retrieve input from your program's users
  • how to perform calculations
