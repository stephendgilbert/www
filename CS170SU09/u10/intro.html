Welcome to Unit 10. Most programs contain errors of one kind or another. In this unit you'll learn how to recognize different kinds of programming errors, and, more importantly, you'll learn how to fix them.

In this unit you'll learn:

  • how to recognize and fix compiler errors.
  • how to recognize and avoid runtime errors.
  • to make your classes more robust using assertions.
  • to add error-checking to your methods with exceptions.
  • how to test your programs using the JUnit4 testing framework.
  • to debug your programs, using the Eclipse debugger.
  • to write programs using arrays of objects.

To the right are the online course materials for this unit. Most of the reading this week is in the online lessons, but you might want to start reading Chapter 18 to get started with exceptions.
