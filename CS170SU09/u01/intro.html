Welcome to CS 170. Before you learn how to program in Java you'll have to spend some time getting prepared. In this first unit, you'll cover some background material and learn about the basic syntax of a Java program.

This week you'll learn:

  • how your computer's memory stores data
  • how your computer's CPU converts data to information
  • about the history of computer programming
  • what Java is, and how it works
  • the basic syntax of a Java program

To the right are the online course materials for this unit. Along with the lessons here, you should also read Chapter 1 in your textbook. Along with the online lectures, there are a few supplemental lessons for setting up Java and Eclipse on your home computer.

In addition to the reading, you'll need to complete the Unit 1 quiz, and have your lab and homework assignments completed and submitted by 5:00 PM on Monday, . By then you should:

  • read each of the online lessons.
  • complete and submit each of the Unit 1 assignments.
  • complete and submit Quiz 1.

NOTE: During the compressed Summer section, we will complete two units each week. The assignments for both units will be due on Monday, before class.

See you online!

Learn about the basic structure and grammar used to write Java programs.
