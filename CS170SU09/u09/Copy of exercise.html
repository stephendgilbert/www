  2. Homework 28 (The Employee Class) : Implement the class Employee. An employee has a name (a string) and a salary (a double). Provide the following constructor and methods:

    public Employee(String name, double initialSalary)
    public String getName()
    public double getSalary()
    public void raiseSalary(double byPercent)

    These methods return the name and salary, and raise the employee's salary by a certain percentage. Here's a sample usage:

    Employee steve = new Employee("Gilbert, Steve", 75000.0);
    steve.raiseSalary(10); // gimme a 10% raise

    Submit the file Employee.java to Web-Cat before the deadline. You can use the Eclipse Submitter plugin, or you can just upload the file through the Web interface. The assignment will be graded on a combination of a set of tests that I've designed (called the reference tests), and 10% on style. The Web-Cat scoring is set to 50 points, which will be scaled down to 5 points.
  3. Homework 29 (The Student Class): Implement a class Student. For the purpose of this exercise, a student has a name and a total quiz score. Supply an appropriate constructor. (Remember, when you create a student, the total quiz score will be 0, so you don't need to pass that as a parameter.) Also, supply the following methods:

    getName()
    addQuiz(int score)
    getTotalScore()
    getAverageScore()

    To compute the last method, remember that you'll also need to keep track of how many quizzes the student actually took.

    Submit the file Student.java to Web-Cat before the deadline. You can use the Eclipse Submitter plugin, or you can just upload the file through the Web interface. The assignment will be graded on a combination of a set of tests that I've designed (called the reference tests), and 10% on style. The Web-Cat scoring is set to 50 points, which will be scaled down to 5 points.
