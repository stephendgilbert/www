Welcome to Unit 4 From 2001's HAL to War Games' WOPPER to Speilberg's AI, popular culture has often portrayed computers as "thinking machines", rather than as glorified calculators.

Computers seem to "think" because, unlike an adding machine, a computer can make decisions and carry out different actions, or repeat a set of actions, based upon the input it receives.

In this first unit on Computer Logic, you'll teach your computer how to make simple decisions and how to repeat actions by learning how to:

  • write user-defined procedures that take parameters
  • write user-defined functions that return values.
  • use the boolean primitive type
  • perform simple decisions using the if statement.
  • design loops to repeat an action
  • use the for loop to generate data

A. Parameters & Functions
Learn how to write procedures that take parameters and write functions that return values.

B. True & False
Learn how to use Java's boolean type and how to compare numbers, objects, and strings.

C. Introducing Selection
Learn how to make decisions with if, else, and the conditional operator.

D. Introducing Loops
Learn about the different types of Java loops, as well as a loop's bound and goal.

E. The for Loop
Learn how to use Java's for statement to create loops that use a counter to control repetition.
