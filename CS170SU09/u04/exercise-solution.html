Exercises and Assignments

This week you'll do exercises writing your own String functions, writing functions using the if statement and writing functions using the for loop. The assignments this week will not use Web-Cat. Instead, we'll be using the JavaBat Web site. The handout linked below in Homework 11 will provide all of the info you need to set up your account.

  1. Homework 11 (Writing Simple String Functions): For this assignment you're going to complete some string exercises at the online homework site JavaBat.com. This is a non-commercial Web site run by Nick Parlante, a CS professor at Stanford. Click this link to learn how to sign up for your JavaBat account and how to make them available for grading. You don't need to do anything special to submit your JavaBat exercises for grading, but you do need to enable the "Teacher Share" feature using my email address.

    For Homework 11, complete the following exercises in the String1 section. For these exercises (which don't involve loops or if statements), you'll use String concatenation, and String functions such as (but not limited to) substring() and length().

    • makeTags (Solution)
    • extraEnd (Solution)
    • firstHalf (Solution)
    • nonStart (Solution)
    • middleTwo (Solution)
    You need to complete the exercises before the deadline. (5 points)
  2. Homework 12 (Simple Logic Problems) : Head on over to http://www.javabat.com and complete the following five problems in the Warmup-1 section:
    • sleepIn (Solution)
    • sumDouble (Solution)
    • diff21 (Solution)
    • makes10 (Solution)
    • nearHundred (Solution)
     Even if you already know how to use nested if statements, for this assignment, I want you to only use simple if statements along with the return statement, like we did for the example problems in lecture. Do not use any of the binary Boolean operators like && or ||. You may use the Boolean not operator (!). (5 points)
  3. Homework 13: Head on over to http://www.javabat.com and complete the following five problems in the String2 section using the for loop. (5 points)
    • catDog (Solution)
    • countCode (Solution)
    • xyzThere (Solution)
    • bobThere (Solution)
    • xyBalance (Solution)
  4. Skim Chapter 5 and Chapter 6, picking up the focusing on the material covered in the online lessons. Complete the Unit 4 Quiz online before the deadline.

Remember that all of these assignments are due before the deadline, so make sure you don't get "caught short".
